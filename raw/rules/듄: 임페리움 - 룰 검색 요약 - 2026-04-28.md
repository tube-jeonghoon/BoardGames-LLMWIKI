## 핵심 요약

듄: 임페리움은 덱빌딩과 일꾼 놓기를 결합한 전략 게임이다. 플레이어는 라운드마다 손패 5장을 받고, 카드의 에이전트 아이콘으로 보드 칸에 에이전트를 보내 자원, 영향력, 병력, 음모 카드를 얻는다. 에이전트를 더 쓰지 않으면 남은 카드를 공개해 설득력으로 새 카드를 사고, 검 아이콘과 배치 병력으로 그 라운드 전투력을 정한다.

승점 10점 이상이 되거나 교전 덱이 비면 해당 라운드 후 게임이 끝난다. 최종 승점이 가장 높은 플레이어가 승리한다.

## 확인한 주요 규칙

- 기본판 기준 인원은 1-4명이다.
- 게임은 여러 라운드로 진행되며 각 라운드는 `라운드 시작 -> 플레이어 턴 -> 전투 -> 메이커스 -> 회수` 5단계다.
- 라운드 시작 때 교전 카드 1장을 공개하고 각 플레이어가 카드 5장을 뽑는다.
- 플레이어 턴에는 에이전트 턴 또는 공개 턴을 한다.
- 에이전트 턴에는 손패 카드 1장을 내고, 카드의 에이전트 아이콘과 맞는 빈 보드 칸에 에이전트를 놓는다.
- 보드 칸 비용은 카드/칸 효과를 받기 전에 즉시 지불할 수 있어야 한다.
- 에이전트 턴에 사용한 카드는 공개 턴에서 다시 공개 효과를 쓰지 않는다.
- 공개 턴에는 남은 손패를 모두 공개해 공개 효과, 설득력, 검을 적용한다.
- 설득력은 그 공개 턴에만 사용하고 저장할 수 없다.
- 전투력은 교전에 배치된 병력 1개당 2 + 공개한 검 아이콘 합계다.
- 교전에 병력이 하나도 없으면 검이 있어도 전투력은 0이다.
- 전투 후 병력은 주둔지로 돌아가지 않고 개인 공급처로 돌아간다.
- 메이커스 단계에는 비어 있는 향신료 생산 칸에 보너스 향신료 1개씩 쌓는다.
- 라운드 종료 시 10점 이상인 플레이어가 있거나 교전 덱이 비면 게임이 끝난다.
- 동점은 향신료, 솔라리, 물, 주둔지 병력 순으로 비교한다.

# 오를로이 - 룰 검색 요약 - 2026-04-30

## 대상

- 오를로이: 프라하 천문시계 / Orloj: The Prague Astronomical Clock
- 기준: 2025년 기본판. 프로모 규칙과 별도 확장은 제외.
- 한국어판 표기: BGG version에는 `오를로이: 프라하 천문시계` Korean edition이 2026년 DiceTree Games/Perro Loko Games 판본으로 등록되어 있음.

## 기본 정보 확인

- Perro Loko Games: 프라하 구시가지의 천문시계 Orloj를 테마로, 플레이어가 황도 구체, 달력의 월, 사도 이미지를 건설한다. 매 턴 시계판에 일꾼을 놓아 주 행동과 보조 행동을 수행하고, 숙련도에 따라 행동을 조정한다. 편차를 강제로 만들면 행동 유연성이 커지지만 수탉이 울기 전 고쳐야 한다.
- Devir: 1-4명, 14세 이상, 120분, 디자이너 Abraham Sanchez Hermida/Paloma J. Pascual, 메커니즘 Set Collection/Worker Placement.
- BGG/BoardGameMatcher 보조: 1-4명, 베스트 2-3인, 120분, 14세 이상, 난이도 3.3/5, BGG Geek Rating 5.8/10, BGG Average Rating 8.1/10, 메커니즘 Action Retrieval, End Game Bonuses, Rondel, Set Collection, Tech Trees / Tech Tracks, Variable Set-up, Worker Placement.
- 보드라이프: 1-4명, 베스트 3인, 추천 1인, 60-120분, 14세 이상, 평점 7.4/10, 난이도 3.33, 진행방식 셋 컬렉션, 일꾼 놓기, 게임 종료 보너스, 변화하는 게임 세팅, 테크 트리/테크 트랙, 보유 액션 수행, 론델.
- TLAMA Games: 1-4명, 120분, 14세 이상, BGG rating 8, complexity 3.6, 메커니즘 End Game Bonuses, Set Collection, Variable Set-up, Worker Placement, Action Retrieval, Rondel, Tech Trees / Tech Tracks.

## 핵심 규칙

- 목표는 오를로이 달력, 황도, 사도 카리용, 작업장/조수/숙련도 등을 통해 조화 점수를 가장 많이 얻는 것이다.
- 게임은 여러 라운드로 진행되며, 황금 수탉 트랙이 라운드의 호흡을 만든다.
- 자기 차례에는 `시계 장치 활성화` 또는 `패스` 중 하나를 선택한다.
- 시계 장치 활성화는 사용 가능한 일꾼이 필요하다.
- 시계 장치 활성화 시 천문 다이얼을 시계방향으로 최소 1칸 돌리고, 필요하면 오를로이 시계판을 반시계방향으로 돌린다.
- 바늘이 가리키는 바깥 링 구역과 시계판 구역이 이번 턴의 두 행동을 정한다.
- 그 지점에 자기 일꾼을 놓고 행동을 수행한다. 다른 플레이어의 일꾼이 있으면 그 일꾼은 소유자에게 돌아간다.
- 숙련도는 다이얼 이동 한계, 패스/수리 효율, 달/화가/조각가 관련 효율을 개선한다.
- 편차를 강제하면 더 유연하게 다이얼/기어를 조정할 수 있지만, 수탉이 울 때 고치지 못하면 감점된다.
- 패스하면 일꾼 회수, 달 활성화, 황금 수탉 전진 등 회복/정비 흐름을 수행한다. 사용 가능한 일꾼이 있어도 패스할 수 있다.
- 추가 행동은 차례 중 가능한 시점에 수행할 수 있다.
- 기본 자원은 페인트, 목재, 철이며, 금과 코인은 특수 자원으로 쓰인다. 보조 출처는 같은 기본 자원 3개를 금 1개로 교환할 수 있다고 설명한다.
- 주요 행동은 자원 생산, 숙련도 상승, 사도 획득, 달 활성화, 화가 이동, 생산/망치 업그레이드, 작업장 확장, 달력/황도 건설, 조각가 활성화다.
- 건설 행동은 망치 디스크를 이동해 건설할 다이얼을 정하고, 행/열에 표시된 자원을 지불해 달력 또는 황도 다이얼을 놓는다.
- 건설할 때 지불한 자원 수와 인접한 자기 일꾼이 있는 다이얼 수가 점수에 영향을 준다.
- 건설에 놓은 일꾼은 게임이 끝날 때까지 그곳에 남으므로, 일꾼 수와 회수 타이밍 관리가 중요하다.
- 황금 수탉이 트랙 끝에 도달하면 현재 플레이어의 턴 후 수탉의 울음이 발생한다. 이때 편차를 고치고 라운드 정비를 한다.
- 보조 출처 기준 종료 조건은 달력이 완성되거나 수탉이 네 번째 울 때다. 공식 제품 설명은 달력이 완성되면 게임이 끝나고 조화 점수가 가장 높은 플레이어가 승리한다고 요약한다.

## 출처별 차이

- 추천 인원: BGG 계열 보조 출처는 베스트 2-3인, 보드라이프는 베스트 3인/추천 1인으로 표시한다.
- 플레이 시간: Devir/BGG 계열은 120분, 보드라이프는 60-120분으로 표시한다.
- 평점: BGG 계열은 Geek Rating 5.8/10과 Average Rating 8.1/10이 분리되어 표시된다. 보드라이프는 7.4/10이다.
- 난이도: BGG 계열 보조 출처 3.3/5, 보드라이프 3.33, TLAMA 3.6으로 약간 다르다.

- 주사위가 비워진 구역은 점수 계산을 유발한다.
- 구성물에는 게임 보드, 플레이어 보드, 대형 타일, 행동 타일, 배 타일, 하버마스터 타일, 프로젝트/물/정원/지구라트/크레이트 타일, 칙령 카드, 도시 카드, 금, 집, 건물, 주사위 등이 포함된다.

### 룰/리뷰 보조 확인

- 턴 구조는 `주사위 선택 -> 건축가 이동 -> 현재 구역 행동 2회 -> 보조자 이동 -> 구역 비었는지 확인` 흐름으로 요약된다.
- 3개 일반 구역에서는 프로젝트 타일 배치, 건물 건설, 정원/물 관련 행동과 가변 행동을 수행한다.
- 항구 구역에서는 집을 놓고 배를 획득해 능력과 점수를 얻는다.
- 지구라트 구역에서는 집/마커를 놓아 지구라트 점수 조건에 영향력을 만든다.
- 구역 주사위가 비면 중간 점수 계산이 발생하고, 게임 종료 후에는 모든 구역을 다시 점수 계산하며 도시 카드를 점수화한다.
- 동점은 금이 많은 플레이어가 우선이고, 그래도 같으면 공동 승리로 확인된다.

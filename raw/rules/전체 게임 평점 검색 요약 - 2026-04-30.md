# 전체 게임 평점 검색 요약 - 2026-04-30

## 범위

현재 위키의 `wiki/games/`와 `wiki/rules/`에 있는 게임 28개의 평점을 보강하기 위해 외부 검색을 수행했다.

## 확인 방식

- BGG XML API는 인증 요구로 직접 조회하지 못했다. BGG 값은 BoardGameMatcher가 공개 페이지에 표시하는 `BGG Rating`을 10점 만점으로 환산해 기록했다.
- `윙스팬: 아메리카 확장`은 BoardGameMatcher 페이지를 찾지 못해 BoardGameGeek 검색 스니펫의 평점 표시를 사용했다.
- 보드라이프 값은 각 보드라이프 게임 페이지의 `게임평점` 메타 설명 또는 검색 스니펫을 확인했다.
- 보드라이프 기존 링크 중 `하다라`는 `game/10007`이 다른 게임으로 연결되어 현재 `game/10861`로 교정했다.

## 주의점

- 평점은 커뮤니티 평가이므로 시간이 지나면 변동될 수 있다.
- BGG의 `Average Rating`과 `Geek Rating`은 서로 다른 값이다. 이번 보강에서 `BGG Rating`은 BoardGameMatcher 공개 페이지의 BGG Rating 표기를 기준으로 했다.
- 보드라이프는 페이지/탭에 따라 크롤링 시점이 다른 검색 스니펫이 보일 수 있으므로, 가능하면 canonical 게임 페이지의 현재 메타 설명을 우선했다.

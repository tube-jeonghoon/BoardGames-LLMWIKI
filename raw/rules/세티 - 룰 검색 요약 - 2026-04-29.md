# 세티 - 룰 검색 요약 - 2026-04-29

## 대상

- SETI: Search for Extraterrestrial Intelligence / 세티: 외계의 지성체를 찾아서
- 기준: Czech Games Edition 2024 기본판. 확장 `SETI: Space Agencies`, 명왕성 프로모, 업그레이드 컴포넌트는 제외.

## 기본 정보 확인

- CGE: 1-4명, 14세 이상, 플레이어당 40분, 2024년 Q4 출시, 디자이너 Tomáš Holek, 장르 Engine Building, Resource Management, Space Exploration.
- BGG: 1-4명, 커뮤니티 1-4인 추천/3인 베스트, 40-160분, 14세 이상, 난이도 3.82/5, 메커니즘 Area Majority / Influence, End Game Bonuses, Income, Multi-Use Cards, Resource to Move 등.
- 보드라이프: 1-4명, 베스트 3인, 추천 2인, 40-160분, 13세 이상, 난이도 3.79.
- 보드라이프 국내 퍼블리셔 표기: DiceTree Games 등.

## 핵심 규칙

- 게임은 5라운드로 진행한다.
- 각 플레이어는 우주 탐사 기관을 맡아 외계 생명체의 흔적을 찾는다.
- 자기 턴에는 주 행동 1개만 수행하고, 자유 행동은 여러 번 할 수 있다.
- 주 행동은 탐사선 발사, 행성 궤도 진입, 행성/위성 착륙, 인근 항성 스캔, 데이터 분석, 카드 효과 사용, 기술 연구, 라운드 패스가 있다.
- 탐사선은 지구에서 출발해 태양계 보드의 행성/위성으로 이동한다.
- 탐사선은 행성 궤도에 남아 궤도선이 되거나, 행성/위성에 착륙해 착륙선이 될 수 있다.
- 스캔은 우주 섹터의 신호를 표시하고 데이터 토큰을 얻는 주요 방법이다.
- 데이터는 자유 행동으로 개인 컴퓨터에 배치하고, 데이터 분석 주 행동으로 보상과 점수를 얻는다.
- 기술 연구는 홍보를 지불해 기술 타일을 얻고 개인 보드 능력을 강화한다.
- 카드는 효과로 사용하거나, 수입 카드로 끼워 넣거나, 자유 행동 아이콘으로 버리는 다목적 카드다.
- 외계 생명체 흔적 3종이 누적되면 게임 중 외계 종족이 발견되고 해당 종족 보드/카드가 열린다.
- 한 라운드에서 패스하면 그 라운드에는 더 이상 턴을 하지 않는다.
- 모든 플레이어가 패스하면 수입을 받고 시작 플레이어를 넘긴 뒤 다음 라운드를 시작한다.
- 5라운드 후 최종 점수: 게임 중 점수 + 금색 점수 타일 + 종료 점수 카드 + 일부 외계 종족 추가 점수.
- 동점이면 공동 승리다.

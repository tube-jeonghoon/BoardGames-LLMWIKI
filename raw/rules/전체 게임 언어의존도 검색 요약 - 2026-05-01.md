# 전체 게임 언어의존도 검색 요약

## 검색 목적

현재 위키에 있는 게임 40개의 `언어의존도` 항목을 보강하기 위해 보드라이프 각 게임 페이지의 현재 `언어의존도` 값을 확인했다.

## 기준

- 값은 보드라이프 게임 페이지의 `언어의존도` 필드 기준이다.
- 보드라이프가 `-`로 표시한 항목은 추정하지 않고 `확인 못함`으로 기록했다.
- 이 문서는 게임 컴포넌트의 실제 텍스트량을 별도로 판정한 것이 아니라, 보드라이프 DB 값을 위키 기본 정보에 반영하기 위한 메타데이터 수집 기록이다.

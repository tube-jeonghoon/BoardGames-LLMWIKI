# 테라포밍마스 질의응답 요약

## 배경

2026-05-02 대화에서 테라포밍마스 기본판 룰 중 초보자가 자주 헷갈릴 만한 생산 단계, 에너지/열의 가치, 카드 점수 계산, 기업상 목록을 정리했다.

## 질문과 정리

### 에너지 자원은 언제 열 자원으로 바뀌나?

에너지 자원은 각 세대의 행동 단계가 모두 끝나고, 모든 플레이어가 패스한 뒤 들어가는 생산 단계에서 열 자원으로 바뀐다.

생산 단계 순서는 다음처럼 이해하면 된다.

1. 사용한 파란 카드 표시를 되돌린다.
2. 남아 있는 에너지 자원 전부를 열 자원으로 옮긴다.
3. 각 생산량에 따라 새 자원을 생산한다.
4. MC는 `TR + MC 생산량`만큼 받는다.

따라서 이번 생산 단계에서 새로 생산한 에너지는 즉시 열로 바뀌지 않는다. 그 에너지는 다음 세대 생산 단계 시작 때 열로 넘어간다.

### 열과 에너지는 온도 상승 말고 쓸 곳이 있나?

열 자원의 기본 용도는 `열 8개 -> 온도 1단계 상승 -> TR +1`이다. 온도가 +8도에 도달하면 이 기본 행동은 더 이상 할 수 없으므로, 열 생산량은 빠르게 가치가 떨어질 수 있다.

일부 카드가 열을 직접 쓰거나 열 생산량과 상호작용할 수 있지만, 기본판에서 열의 범용성은 낮다. 온도가 2-3칸밖에 남지 않았다면 새 열 생산 카드는 남은 세대 수와 실제 온도 상승 횟수를 보고 신중하게 평가한다.

에너지 자원은 생산 단계에서 열로 바뀌는 중간 자원이지만, 열보다 쓰임새가 조금 더 있다.

- 일부 파란 카드 행동이 에너지를 비용으로 쓴다.
- 일부 카드, 특히 도시/건물 계열 카드는 에너지 생산량 감소를 비용처럼 요구한다.
- 에너지 사용 카드가 없고 에너지 생산량을 요구하는 카드를 낼 계획도 없다면, 결국 열로 넘어가므로 온도 마감 후 가치는 낮아진다.

### 카드 오른쪽 아래 점수는 언제 계산하나?

카드 오른쪽 아래의 VP는 게임 중 즉시 점수 트랙에 반영하지 않고, 게임 종료 후 최종 점수 계산 때 더한다.

카드 점수는 크게 두 종류다.

- 고정 점수: `1 VP`, `2 VP`처럼 카드 자체에 표시된 점수
- 조건 점수: 동물, 미생물, 목성 태그 등 특정 조건/자원에 따라 계산하는 점수

둘 다 게임 종료 후 `TR, 녹지, 도시 인접 녹지, 업적, 기업상`을 계산한 뒤 카드 점수로 합산한다. 단, 카드 효과로 전역 지표를 올려 TR이 오르는 것은 해당 효과를 해결할 때 즉시 반영한다.

### 기본 맵 기업상은 무엇이 있나?

기본판 기본 맵 기업상은 5개다.

| 기업상 | 조건 |
| --- | --- |
| 지주 / Landlord | 보드 위에 가진 타일 수가 가장 많은 플레이어 |
| 은행가 / Banker | MC 생산량이 가장 높은 플레이어 |
| 과학자 / Scientist | 과학 태그가 가장 많은 플레이어 |
| 열 전문가 / Thermalist | 열 자원 수가 가장 많은 플레이어 |
| 광부 / Miner | 철강 + 티타늄 자원 수가 가장 많은 플레이어 |

기업상은 후원한 사람이 자동으로 점수를 받는 것이 아니다. 누군가 기업상을 후원하면 그 분야의 점수 경쟁이 열리고, 게임 종료 때 실제 1등이 5점, 2등이 2점을 받는다. 기본판에서는 기업상도 최대 3개까지만 후원된다.

# 🎲 하다라

## 🧩 한줄 요약

카드를 사거나 버리면서 문명의 소득, 군사력, 식량, 문화, 점수 능력을 키우고, 3시대 종료 후 가장 높은 점수를 얻는 문명 테마 카드 드래프팅 게임이다.

## 📌 기본 정보

| 항목 | 내용 |
| --- | --- |
| 최소인원 | 2명 |
| 최대인원 | 5명 |
| 추천인원 | BGG: 추천/베스트 세부값 직접 확인 못함 / 보드라이프: 베스트 4인, 추천 3인 |

> 확인 출처: BGG `Hadara`, 보드라이프 `하다라`

출처별 차이: 보드라이프는 베스트 4인과 추천 3인을 제공한다. BGG 페이지는 확인했지만 동적 페이지/API 제한으로 추천인원 세부값을 직접 확인하지 못했다. 플레이 시간은 Z-Man과 BGG 보조 정보가 45-60분, 보드라이프가 45-60분으로 표기한다.

## 🌍 세계관 / 테마

플레이어는 여러 시대를 거치며 문명을 발전시킨다. 각 색 카드가 문명의 서로 다른 분야를 나타내며, 인물과 발전 요소를 받아들여 소득, 군사력, 식량, 문화, 점수 능력을 키운다.

## 🏆 승리 조건

3시대가 끝난 뒤 카드 점수, 식민지, 조각상, 금/은 메달, 은화, 트랙 점수 등을 모두 합산해 가장 높은 점수를 얻은 플레이어가 승리한다.

동점이면 남은 은화가 더 많은 플레이어가 승리한다. 그래도 같으면 공동 승리한다.

## 🔁 전체 게임 흐름

하다라는 `카드를 고른다 -> 구입하거나 버려 돈을 얻는다 -> 문명 능력치를 올린다 -> 시대 종료 보상을 받는다 -> 최종 점수를 얻는다`의 흐름으로 진행된다.

각 시대에는 먼저 A 페이즈를 통해 비공개 카드 더미에서 카드를 고르고, 이후 B 페이즈에서 A 페이즈 중 공개로 버려진 카드들을 고른다. 카드는 사면 능력치와 점수를 제공하지만 돈이 필요하고, 버리면 즉시 돈을 준다. 카드가 늘어날수록 식량도 충분히 확보해야 한다.

## 준비 정보

### 주요 구성물

- 게임 보드 1개
- 문명 보드 5개
- 시작 플레이어 마커
- 카드 162장
- 식민지 타일
- 조각상 타일
- 은화
- 각 플레이어별 문장 타일과 마커

### 카드 색의 감각

- 노란색: 소득과 돈 운용
- 빨간색: 군사력과 식민지
- 파란색: 문화/조각상
- 초록색: 식량
- 보라색: 점수와 특수 보상

카드 색별 역할은 판본/카드 구성에 따라 세부 효과가 다를 수 있으므로, 실제 카드는 아이콘을 우선 확인한다.

## 🎮 턴 구조

게임은 3시대로 진행된다. 각 시대는 아래 순서로 진행한다.

1. A 페이즈
2. 시대 중간 수입/정산
3. B 페이즈
4. 시대 종료 수입/정산

A 페이즈와 B 페이즈에서 플레이어는 차례대로 카드를 선택하고, 선택한 카드를 구입하거나 버린다.

## 🧱 페이즈별 설명

### A 페이즈

1. 시작 플레이어부터 차례로, 자신의 문장 타일이 가리키는 색 카드 더미에서 카드 2장을 뽑는다.
2. 2장 중 1장을 선택한다.
3. 선택하지 않은 1장은 같은 색의 공개 버린 카드 칸에 놓는다.
4. 선택한 카드는 구입하거나 버린다.
5. 문장 타일을 다음 색으로 이동한다.

모든 플레이어가 각 색 카드 더미를 한 번씩 사용하면 A 페이즈가 끝난다.

### 카드 구입 또는 버리기

선택한 카드는 둘 중 하나로 처리한다.

- 구입: 카드 비용을 내고 자신의 문명 보드 아래에 색깔별로 배치한다.
- 버리기: 카드를 버리고 시대에 따른 은화를 받는다.

구입 비용은 카드 왼쪽 위 비용에서, 이미 보유한 같은 색 카드 수만큼 할인된다. 비용은 0 미만이 될 수 없다.

### 시대 중간 정산

A 페이즈가 끝나면 문명 보드의 소득 트랙에 따라 은화를 받는다.

그 뒤 군사력으로 식민지를 정복할 수 있는지 확인한다. 식민지는 약탈하거나 편입할 수 있으며, 선택에 따라 즉시 은화 또는 장기 보상을 얻는다.

문화 수치로 조각상을 만들 수 있다. 조각상은 보너스 능력이나 점수를 제공한다.

마지막으로 식량을 확인한다. 보유 카드 수가 식량 수치보다 많으면 초과한 카드만큼 버려야 한다.

### B 페이즈

B 페이즈에서는 A 페이즈에서 공개로 버려진 카드들을 사용한다.

1. 시작 플레이어부터 차례로, 공개 버린 카드 더미 중 하나에서 카드 1장을 선택한다.
2. 선택한 카드는 구입하거나 버린다.
3. 모든 플레이어가 정해진 횟수만큼 카드를 선택하면 B 페이즈가 끝난다.

### 시대 종료 정산

B 페이즈가 끝나면 A 페이즈 후와 비슷하게 소득, 식민지, 조각상, 식량 확인을 처리한다.

그 뒤 조건을 만족하면 은 메달 또는 금 메달을 구입할 수 있다. 메달은 최종 점수에 크게 기여한다.

3시대가 끝나지 않았다면 다음 시대를 준비한다. 3시대가 끝났다면 최종 점수 계산으로 간다.

### 최종 점수 계산

아래 점수를 모두 더한다.

1. 카드에 표시된 점수
2. 식민지 점수
3. 조각상 점수
4. 은 메달 점수
5. 금 메달 점수
6. 남은 은화 5개당 1점
7. 문명 보드의 점수 트랙 값

## ⚠️ 자주 헷갈리는 규칙 / 예외 상황

- A 페이즈에서 선택하지 않은 카드는 그냥 버리는 것이 아니라, 같은 색 공개 버린 카드 칸에 놓아 B 페이즈 선택지가 된다.
- B 페이즈에서 가져온 카드도 똑같이 구입하거나 버릴 수 있다.
- 같은 색 카드 수에 따른 할인은 이미 구입해 보유 중인 카드만 본다.
- 식량 부족으로 카드를 버릴 때는 어떤 카드를 버릴지 신중히 고른다. 버린 카드는 능력치와 점수를 더 이상 제공하지 않는다.
- 식민지는 군사력 조건을 만족해야 가져올 수 있다.
- 메달 구입은 시대 종료 정산에서 조건과 비용을 확인한다.
- 판본/룰북에 따라 일부 용어 번역이 다를 수 있음. 아이콘과 단계 순서를 우선한다.

## 자주 하는 실수

- 돈이 부족한데 좋은 카드만 계속 사려고 한다. 하다라는 카드를 버려 돈을 얻는 선택도 핵심이다.
- 초록색 식량을 방치해 시대 종료 때 카드를 버린다.
- A 페이즈에서 버린 카드가 B 페이즈 선택지가 된다는 점을 잊고 상대에게 좋은 카드를 열어준다.
- 할인 계산 때 방금 살 카드를 포함한다. 할인은 이미 갖고 있는 같은 색 카드 수를 기준으로 한다.
- 점수 카드만 모으고 식민지, 조각상, 메달 점수를 놓친다.

## 💡 초보자 팁

- 초반에는 소득과 식량을 안정적으로 확보한다. 돈이 없으면 좋은 카드를 살 수 없고, 식량이 부족하면 산 카드를 버리게 된다.
- 빨간색 군사력은 식민지 점수로 이어지고, 파란색 문화는 조각상 보상으로 이어진다. 한쪽만 보지 말고 시대 종료 보상까지 계산한다.
- A 페이즈에서 버리는 카드는 B 페이즈에 공개된다. 내가 못 살 카드라도 다음 플레이어에게 너무 좋은 카드인지 확인한다.
- 같은 색 카드를 여러 장 모으면 할인 덕분에 이후 카드 구입이 쉬워진다.
- 메달은 최종 점수 비중이 크므로, 2-3시대에는 조건과 비용을 미리 준비한다.

## 모순/변경점

- 한국어 공식 룰북 PDF는 이번 검색에서 확인하지 못했다.
- BGG 추천인원/베스트 세부값은 BGG 페이지/API 제한으로 직접 확인하지 못했다. 보드라이프의 추천인원 값은 별도로 기록했다.
- 메커니즘 표기는 출처마다 분류 체계가 달라 완전히 일치하지 않는다.

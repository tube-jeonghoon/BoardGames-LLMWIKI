# 🎲 윙스팬: 아메리카 확장

## 🧩 한줄 요약

벌새 미니 카드가 개인 매트와 중앙 정원을 오가며, 서식지 행동 끝에 추가 보상과 벌새 트랙 점수를 만드는 2026년 확장이다.

| 난이도 | BGG: 0.00/5(투표 부족) / 보드라이프: 미표기 / Stonemaier: 중간 난이도 |

> 확인 출처: BGG `Wingspan: Americas Expansion`, 보드라이프 `윙스팬 : 아메리카 확장`

출처별 차이: 플레이 시간은 Stonemaier가 40-70분, BGG와 보드라이프가 75-90분으로 다르게 표시한다. 난이도는 BGG 투표가 아직 부족하고 보드라이프도 미표기다.

## 🌍 세계관 / 테마

멕시코, 중앙아메리카, 남아메리카, 카리브해의 새들이 추가된다. 특히 라틴아메리카에서 큰 비중을 차지하는 벌새를 별도 미니 카드 시스템으로 구현한다.

## 🏆 승리 조건

기본 최종 점수 구조에 벌새 트랙 점수가 추가된다. 벌새는 일반 새와 다른 방식으로 점수를 만든다.

벌새 트랙의 각 열에서 자기 토큰 위치에 따라 점수를 얻거나 잃을 수 있다. 정확한 점수값은 실제 벌새 트랙에 인쇄된 값을 따른다.

## 🔁 전체 게임 흐름

기본 윙스팬 흐름에 벌새 행동이 추가된다.

`서식지 행동을 한다 -> 그 줄의 갈색 능력을 오른쪽에서 왼쪽으로 처리한다 -> 벌새 공간에 도착한다 -> 비어 있으면 벌새를 끌어오고 보상을 받는다 -> 이미 있으면 벌새를 정원으로 돌려보내고 벌새 트랙을 전진한다`의 흐름이다.

새 놓기 행동에는 벌새 행동이 붙지 않는다. 먹이 얻기, 알 낳기, 새 카드 뽑기처럼 행동 큐브가 서식지 줄을 따라 이동하는 행동에서 벌새 공간을 처리한다.

## 준비 정보

### 주요 구성물

- 아메리카 새 카드 111장
- 벌새 미니 카드 40장
- 벌새 정원 보드
- 플레이어 매트 오버레이
- 벌새 트랙
- 벌새 토큰
- 보너스 카드 8장
- 양면 라운드 목표 타일 5개
- 넥타르 토큰
- 참조 타일
- 점수 기록지
- 오토마 구성물

### 셋업 요약

1. 아메리카 새 카드를 기존 새 덱에 섞는다.
2. 아메리카 보너스 카드와 라운드 목표 타일을 기존 구성물에 섞는다.
3. 벌새 정원 보드를 중앙에 둔다.
4. 벌새 미니 카드를 섞어 덱을 만들고, 정원에 벌새 5장을 공개한다.
5. 각 플레이어는 매트 오버레이와 벌새 트랙, 벌새 토큰을 준비한다.
6. 각 플레이어는 시작 구성으로 새 카드, 음식, 보너스 카드를 기본판처럼 선택한다.
7. 아메리카 규칙을 사용할 때 시작 넥타르와 넥타르 토큰 처리는 해당 룰북/참조 타일을 따른다.

## 🎮 턴 구조

자기 차례의 큰 행동은 기본판과 같다.

1. 새 놓기
2. 먹이 얻기
3. 알 낳기
4. 새 카드 뽑기

벌새 행동은 2-4번 서식지 행동이 끝난 뒤 처리한다. 새 놓기 행동 뒤에는 벌새 행동을 하지 않는다.

## 🧱 페이즈별 설명

### 1. 서식지 행동 처리

먹이 얻기, 알 낳기, 새 카드 뽑기 중 하나를 선택한다. 기본 행동을 처리하고, 그 줄의 갈색 능력을 오른쪽에서 왼쪽으로 처리한다.

그 뒤 매트 오버레이의 벌새 공간을 확인한다.

### 2. 벌새 공간이 비어 있을 때: 벌새 끌어오기

해당 서식지의 벌새 공간이 비어 있으면 벌새를 끌어온다.

1. 벌새 정원에 공개된 벌새 또는 벌새 덱에서 벌새 1장을 선택한다.
2. 그 벌새를 해당 서식지의 빈 벌새 공간에 놓는다.
3. 벌새 카드 왼쪽 아래의 보상을 받는다.

정원에서 벌새를 가져가 생긴 빈칸은 턴 종료 때 보충한다.

### 3. 벌새 공간에 벌새가 있을 때: 벌새 돌려보내기

해당 서식지의 벌새 공간에 이미 벌새가 있으면 그 벌새를 정원으로 돌려보낸다.

1. 개인 매트의 벌새를 정원으로 옮긴다.
2. 보통 정원의 다른 벌새 위를 덮어 놓는다.
3. 돌려보낸 벌새의 그룹 또는 덮은 벌새의 그룹 중 하나를 골라 자신의 벌새 트랙 해당 열을 전진한다.

벌새 트랙에서 특정 아이콘에 도착하면 추가 벌새 행동을 할 수 있다. 이 추가 벌새 행동은 원하는 서식지에서 처리한다.

### 4. 벌새 정원 관리

벌새 정원은 새 카드 트레이와 비슷하게 관리한다.

- 비어 있는 정원 칸은 플레이어 턴 종료 때 보충한다.
- 라운드 종료 때 정원을 정리하고 새로 공개한다.
- 벌새 덱이 떨어지면 버린 벌새를 섞어 새 덱을 만든다.

### 5. 벌새의 카드 성격

아메리카 확장의 미니 벌새 카드는 새로 센다. 보호구역에 있는 벌새는 해당 서식지의 새로 본다.

다만 아래 제한이 있다.

- 벌새에는 알을 놓을 수 없다.
- 벌새 위에 먹이를 저장할 수 없다.
- 벌새 밑에 카드를 끼워 넣을 수 없다.
- 벌새 공간은 벌새 전용이다. 일반 새를 그 칸에 놓을 수 없다.
- 벌새는 점수 0점, 음식 비용 0, 알 수용량 0으로 본다.
- 날개 길이 조건에서는 모든 아메리카 미니 벌새를 가장 작은 새처럼 취급한다.
- 이름을 보는 보너스 카드에는 벌새 이름이 적용될 수 있다.

듀엣 모드와 섞어 할 때 벌새를 끌어오는 행동은 새를 `놓는` 행동이 아니므로 듀엣 지도에 토큰을 놓지 않는다.

### 6. 게임 종료와 벌새 점수

4라운드 종료 후 기본 최종 점수를 계산하고, 벌새 트랙 점수를 더한다.

벌새 트랙은 그룹별로 전진한 정도에 따라 점수를 준다. 일부 위치는 감점일 수 있으므로, 게임 종료 전에 각 열의 토큰 위치를 확인한다.

## ⚠️ 자주 헷갈리는 규칙 / 예외 상황

- 벌새 행동은 새 놓기 뒤에 하지 않는다.
- 벌새 공간이 비어 있으면 벌새를 끌어오고, 이미 있으면 돌려보낸다.
- 벌새는 새로 세지만 알/저장 먹이/끼워 넣기 대상이 아니다.
- 벌새를 끌어오는 것은 새를 놓는 행동이 아니다.
- 아시아 듀엣 모드와 섞어도 벌새를 끌어올 때 듀엣 토큰을 놓지 않는다.
- 벌새 트랙 점수는 트랙에 인쇄된 값을 따른다. 일부 위치는 감점일 수 있다.
- 정원 빈칸 보충은 턴 종료 때 한다.

## 자주 하는 실수

- 새 놓기 행동 뒤에도 벌새 행동을 한다.
- 벌새에 알을 낳거나 먹이를 저장한다.
- 벌새 공간에 일반 새를 놓는다.
- 벌새를 끌어오면서 듀엣 토큰을 놓는다.
- 벌새 정원 빈칸을 즉시 보충할지, 턴 종료 때 보충할지 헷갈린다.
- 벌새 트랙의 감점/득점 위치를 최종 점수에서 빼먹는다.

## 💡 초보자 팁

- 아메리카 확장은 윙스팬 기본 흐름을 이미 아는 플레이어에게 추천한다.
- 벌새 행동은 거의 매 서식지 행동마다 발생하므로, 벌새 공간이 비었는지 차례 시작 전에 확인한다.
- 초반에는 벌새를 끌어와 즉시 보상을 받는 것이 자원 흐름을 빠르게 만든다.
- 중후반에는 벌새를 돌려보내 트랙 점수를 올리는 계획을 세운다.
- 4-5인에서는 플레이 시간이 늘 수 있다. 룰북의 짧은 게임 변형처럼 시작 행동 큐브를 1개 줄이는 방식도 고려할 수 있다.

## 모순/변경점

- 아메리카 확장은 2026년 출시 확장이다. BGG와 보드라이프의 난이도/추천 인원 투표가 아직 부족해 안정적인 커뮤니티 값이 없다.
- 플레이 시간은 Stonemaier 40-70분, BGG/보드라이프 75-90분으로 차이가 있다.
- 한국어판 공식 룰북은 이번 검색에서 직접 확인하지 못했다. 국내판 용어는 실제 룰북과 다를 수 있음.

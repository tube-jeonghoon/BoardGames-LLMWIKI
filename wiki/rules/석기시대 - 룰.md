# 🎲 석기시대

## 🧩 한줄 요약

부족원을 장소에 배치해 식량과 자원을 얻고, 도구·농업·인구·문명 카드·건물을 조합해 가장 많은 점수를 얻는 일꾼 놓기 게임.

## 📌 기본 정보

| 항목 | 내용 |
| --- | --- |
| 최소인원 | Hans im Glück: 2명 / BGG 계열: 2명 / 보드라이프: 2명 |
| 최대인원 | Hans im Glück: 4명 / BGG 계열: 4명 / 보드라이프: 4명 |
| 추천인원 | BGG 계열: 베스트 4인 / 보드라이프: 베스트 4인, 추천 3인 |
| 메커니즘 | 일꾼 놓기, 주사위 굴림, 세트 컬렉션, 계약/건물 구매, 라운드 종료 먹이기, 최종 보너스 |
| 난이도 | BGG 계열: 2.46-2.50/5 / 보드라이프: 2.46 또는 2.17 |
| 플레이 시간 | Hans im Glück/BGG 계열/보드라이프: 60-90분 |
| 평점 | BGG 계열: 약 7.5-7.6 / 보드라이프: 7.5 |
| 언어의존도 | 보드라이프: 없음 |

> 확인 출처: Hans im Glück `Stone Age`, Hans im Glück 공식 룰북 링크, RulesPal 룰북 전사본, BGG/BoardGameOracle, 보드라이프 `석기시대`.

## 🌍 세계관 / 테마

플레이어는 석기시대 부족을 이끌어 나무를 베고, 점토와 돌을 캐고, 강에서 금을 씻어 얻는다. 도구와 농업을 발전시키고, 부족을 늘리며, 건물과 문명 카드를 통해 부족을 성장시킨다.

## 🏆 승리 조건

게임 종료 후 점수가 가장 높은 플레이어가 승리한다.

동점이면 농업 트랙 위치, 도구 총합, 보유 부족원 수의 합이 더 큰 플레이어가 승리한다.

## 🔁 전체 게임 흐름

게임은 정해진 라운드 수가 아니라 종료 조건이 발생할 때까지 여러 라운드로 진행된다.

각 라운드는 3단계다.

1. 부족원 배치
2. 배치한 부족원 행동 해결
3. 부족 먹이기

## 🧱 셋업 요약

1. 게임판을 가운데 놓는다.
2. 식량 토큰을 값별로 나누어 사냥터에 놓는다.
3. 목재 28개를 숲에, 점토 18개를 점토 구덩이에, 석재 12개를 채석장에, 금 10개를 강에 놓는다.
4. 도구 타일을 값별로 나누어 마을의 도구 제작자 공간에 놓는다.
5. 문명 카드를 섞고 4장을 공개한다.
6. 건물 타일을 섞어 7장짜리 더미를 만든다. 플레이어 수만큼 더미를 사용하고 각 더미 맨 위를 공개한다.
7. 각 플레이어는 색을 고르고 점수 마커를 0점, 농업 마커를 0칸에 놓는다.
8. 각 플레이어는 개인판, 부족원 5명, 식량 12를 받는다.
9. 남은 부족원 5명씩은 공급처에 둔다.
10. 가장 어린 플레이어가 시작 플레이어가 된다.

## 🎮 1단계: 부족원 배치

시작 플레이어부터 시계방향으로, 자기 차례마다 한 장소를 골라 그 장소에 허용되는 수만큼 부족원을 놓는다.

중요한 제한은 다음과 같다.

- 아직 놓을 수 있는 부족원이 있으면 배치를 패스할 수 없다.
- 같은 라운드에 이미 자기 부족원이 있는 장소에는 추가로 놓을 수 없다.
- 장소마다 들어갈 수 있는 부족원 수가 다르다.

## 📍 장소별 배치 제한

| 장소 | 배치 제한 |
| --- | --- |
| 도구 제작자 | 라운드당 1명 |
| 오두막 | 같은 플레이어 부족원 정확히 2명 |
| 밭 | 라운드당 1명 |
| 사냥터 | 제한 없음 |
| 숲/점토 구덩이/채석장/강 | 장소마다 최대 7명 |
| 문명 카드 | 카드마다 1명 |
| 건물 타일 | 건물마다 1명 |

## ⚙️ 2단계: 행동 해결

모든 플레이어가 더 이상 부족원을 배치할 수 없으면 시작 플레이어부터 행동을 해결한다. 자기 차례에는 자기 부족원이 있는 장소를 원하는 순서로 해결한다. 행동을 해결한 부족원은 개인판으로 돌아온다.

### 도구 제작자

도구 타일을 얻거나 강화한다. 도구가 없으면 값 1 도구를 받는다. 이후에는 빈 도구 칸을 채우고, 모든 칸이 찬 뒤에는 도구 값을 2, 3, 4로 올린다.

도구는 주사위 굴림 결과에 더할 수 있다. 각 도구는 라운드당 한 번만 사용할 수 있고, 일부 값만 나누어 쓰지 않는다. 사용한 도구는 90도 돌려 이번 라운드에 썼음을 표시한다.

### 오두막

오두막에 놓은 부족원 2명을 회수하고, 공급처에서 자기 부족원 1명을 가져와 개인판에 추가한다. 이후 라운드부터 사용할 수 있다.

### 밭

농업 트랙을 1칸 전진한다. 농업 트랙은 먹이기 단계 시작 때 받는 식량 수를 늘린다.

### 사냥터

사냥터에 놓은 부족원 수만큼 주사위를 굴린다. 주사위 합에 사용한 도구 값을 더한 뒤 2로 나누고, 나머지는 버린다. 그 몫만큼 식량을 받는다.

### 숲, 점토 구덩이, 채석장, 강

각 장소에 놓은 부족원 수만큼 주사위를 굴리고, 도구를 더할 수 있다. 주사위 합을 장소별 숫자로 나눈 몫만큼 해당 자원을 받는다.

| 장소 | 자원 | 나눗값 |
| --- | --- | --- |
| 숲 | 목재 | 3 |
| 점토 구덩이 | 점토 | 4 |
| 채석장 | 석재 | 5 |
| 강 | 금 | 6 |

자원은 문명 카드와 건물 타일을 얻는 데 사용한다. 자원이 부족하면 대체물을 사용해도 된다.

### 문명 카드

부족원이 놓인 문명 카드를 가져오고, 카드 위에 표시된 수만큼 원하는 자원을 지불한다. 식량은 자원이 아니므로 비용으로 낼 수 없다.

문명 카드는 보통 즉시 효과와 게임 종료 점수 효과를 제공한다. 카드는 자기 개인판에 뒷면으로 놓고 언제든 확인할 수 있다.

비용을 낼 수 없거나 내고 싶지 않으면 카드를 가져오지 않고 부족원만 회수한다.

### 건물 타일

부족원이 놓인 건물 타일을 가져오고, 표시된 자원을 지불한다. 즉시 건물에 표시된 점수만큼 점수 트랙을 전진한다.

일부 건물은 자원 종류를 선택할 수 있지만 개수는 정해져 있다. 또 일부 건물은 최대 7개까지 자원 수와 종류를 고를 수 있다. 이 경우 자원별 점수 가치로 직접 계산해 점수를 얻는다.

## 🍖 3단계: 부족 먹이기

모든 플레이어가 행동을 끝내면 부족을 먹인다.

1. 먼저 자기 농업 트랙 수치만큼 식량을 받는다.
2. 자기 개인판에 있는 부족원 1명당 식량 1을 지불한다.
3. 남은 식량은 다음 라운드로 가져간다.

식량이 부족하면 다음 순서로 처리한다.

1. 가진 식량은 모두 지불해야 한다.
2. 원하면 부족한 만큼 자원을 1:1로 지불할 수 있다.
3. 자원을 지불하지 않거나 지불할 수 없으면 즉시 10점을 잃는다. 부족한 식량 수와 관계없이 항상 10점이다.

## 🔄 새 라운드 준비

먹이기가 끝나면 시작 플레이어 토큰을 왼쪽 플레이어에게 넘긴다.

남은 문명 카드는 오른쪽으로 밀고, 빈 칸을 새 카드로 채운다. 도구 타일은 다시 사용할 수 있도록 원래 방향으로 돌린다.

## 🧮 종료와 최종 점수

게임은 두 경우 중 하나로 끝난다.

- 새 라운드 시작 때 문명 카드 4칸을 채울 카드가 부족하다. 이 경우 새 라운드를 시작하지 않고 즉시 끝난다.
- 건물 더미 중 하나가 비었다. 이 경우 현재 라운드를 먹이기까지 끝내고 게임이 끝난다.

최종 점수는 다음을 더한다.

- 초록 배경 문명 카드: 서로 다른 종류 개수의 제곱 점수. 여러 세트를 만들 수 있다.
- 모래색 문명 카드:
  - 농부 수 x 농업 트랙 위치
  - 도구 제작자 수 x 도구 총합
  - 건축가 수 x 건물 타일 수
  - 주술사 수 x 보유 부족원 수
- 남은 자원: 식량을 제외하고 자원 1개당 1점

## 👥 2-3인 게임 변경

2-3인 게임에서는 일부 장소 점유 제한이 추가된다.

- 2인/3인: 도구 제작자, 오두막, 밭 중 한 라운드에 2곳만 점유될 수 있다. 나머지 1곳은 비어 있어야 한다.
- 3인: 숲, 점토 구덩이, 채석장, 강은 각 장소마다 최대 2명의 플레이어만 들어갈 수 있다.
- 2인: 숲, 점토 구덩이, 채석장, 강은 각 장소마다 최대 1명의 플레이어만 들어갈 수 있다.

그 외 규칙은 동일하다.

## ⚠️ 자주 헷갈리는 규칙 / 예외 상황

- 같은 라운드에 같은 장소로 자기 부족원을 나눠서 여러 번 보낼 수 없다.
- 오두막은 반드시 자기 부족원 2명을 동시에 놓아야 한다.
- 사냥터에는 제한 없이 들어갈 수 있지만, 다른 자원 장소는 최대 7명 제한이 있다.
- 문명 카드 비용은 카드 위치 위에 표시된 수만큼 원하는 자원으로 낸다.
- 식량은 자원이 아니므로 문명 카드나 건물 비용으로 낼 수 없다.
- 도구는 한 라운드에 한 번만 사용하고, 값 일부만 쪼개서 저장할 수 없다.
- 먹이기 실패 시 부족한 식량이 1개든 5개든 감점은 10점이다.

## 자주 하는 실수

- 먹이기 전에 농업 트랙만큼 식량 받는 것을 잊는다.
- 도구를 여러 번 쓰거나, 값 3 도구를 1만 쓰고 2를 남겨 둔다.
- 2-3인 게임의 장소 제한을 4인과 똑같이 처리한다.
- 건물 더미가 빈 즉시 게임을 끝낸다. 실제로는 현재 라운드를 먹이기까지 끝낸다.
- 문명 카드 더미가 부족한 종료 조건에서도 새 라운드를 시작한다.

## 💡 초보자 팁

- 사람 수를 늘리면 행동 수가 늘지만 먹일 식량도 늘어난다. 오두막만 무작정 가면 감점 압박이 커진다.
- 도구는 비싼 자원 장소에서 실패를 줄이는 보험이다.
- 농업 트랙은 장기적으로 먹이기 부담을 줄여 주지만, 점수로 직접 바꾸려면 농부 문명 카드와 맞아야 한다.
- 건물은 즉시 점수라 안정적이고, 문명 카드는 최종 점수 잠재력이 크다.
- 초록 문명 카드는 같은 종류만 모으기보다 서로 다른 종류 세트를 만드는 것이 중요하다.

## 모순/변경점

- 공식 Hans im Glück 페이지는 2-4인, 60-90분, 10세 이상으로 표기한다.
- 보드라이프에서는 난이도 2.46과 2.17이 서로 다른 페이지 노출에서 확인된다. 최신/상세 페이지 대조가 필요하다.
- BGG 직접 페이지는 웹 도구에서 본문을 확인하지 못해 BoardGameOracle 등 BGG 계열 보조 정보를 함께 사용했다.
- 한국어판 공식 용어는 직접 대조하지 못했다. 이 문서는 기능 중심 번역어를 사용한다.
- 확장 규칙과 어린이판 [[내 첫 석기시대]]는 제외했다.

# 7 원더스와 7 원더스 대결 차이

## 한줄 요약

[[7 원더스]]는 3-7명이 동시에 손패를 돌리며 도시를 키우는 다인 카드 드래프팅 게임이고, [[7 원더스 대결]]은 2명만 하는 공개 카드 선택/견제 게임이다. 테마와 카드 다중 사용 감각은 비슷하지만, 실제 플레이 감각은 꽤 다르다.

## 핵심 차이

| 항목     | 7 원더스                       | 7 원더스 대결                              |
| ------ | --------------------------- | ------------------------------------- |
| 인원     | 2판 기준 3-7인                  | 2인 전용                                 |
| 카드 선택  | 손패에서 1장 선택 후 남은 카드를 이웃에게 넘김 | 겹쳐 놓인 카드 중 덮이지 않은 카드 1장을 선택           |
| 턴 진행   | 모두 동시에 선택/공개                | 번갈아 한 명씩 선택                           |
| 승리 방식  | 3시대 종료 후 점수 합산              | 군사/과학 즉시 승리 또는 3시대 후 점수               |
| 상호작용   | 양옆 이웃과 자원 거래, 군사 비교, 카드 차단  | 카드 공개 순서, 군사 트랙, 과학 기호, 자원 가격으로 직접 견제 |
| 자원 구매  | 양옆 이웃에게 코인을 지불              | 부족분을 은행에서 구매하되 상대 생산량에 따라 비싸짐         |
| 군사     | 각 시대 말에 양옆 이웃과 비교해 토큰 획득    | 중앙 트랙을 밀어 상대 수도에 도달하면 즉시 승리           |
| 과학     | 기호 세트와 같은 기호 묶음으로 점수        | 서로 다른 과학 기호 6개면 즉시 승리                 |
| 불가사의   | 각자 불가사의 보드의 단계를 건설          | 불가사의 카드 4장씩 선택, 게임 중 총 7개만 건설 가능      |
| 플레이 감각 | 빠르고 동시 진행, 테이블 전체 흐름을 읽음    | 더 전술적이고 계산적, 상대 한 명의 다음 선택을 강하게 의식    |

## 어떤 사람에게 맞나

- 여러 명이 빠르게 동시에 플레이하고, 카드 드래프트와 점수 엔진을 즐기고 싶으면 [[7 원더스]]가 맞다.
- 거의 2인으로만 하고, 상대가 무엇을 열어줄지/막을지 계산하는 대결감을 원하면 [[7 원더스 대결]]이 맞다.
- 7 원더스 대결은 7 원더스의 2인 변형이 아니라 별도 독립 게임이다.

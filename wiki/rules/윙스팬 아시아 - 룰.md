# 🎲 윙스팬 아시아

## 🧩 한줄 요약

아시아 조류 카드 확장이면서, 2인 듀엣 모드와 6-7인 플록 모드를 추가하는 독립형/확장 겸용 윙스팬이다.

## 📌 기본 정보

| 항목     | 내용                                                                  |
| ------ | ------------------------------------------------------------------- |
| 최소인원   | 단독 게임: 1명 / 기본판 결합 플록 모드: 6명                                        |
| 최대인원   | 단독 게임: 2명 / 기본판 결합 플록 모드: 7명                                        |
| 추천인원   | 보드라이프: 베스트 2인, 추천 1인 / Stonemaier: 1-7인 제품 표기, 단독 1-2인 및 플록 6-7인 구조 |
| 메커니즘   | 카드 드래프팅, 주사위 굴림, 핸드 관리, 셋 컬렉션, 태블로 빌딩, 듀엣 지도 영향력                    |

> 확인 출처: BGG `Wingspan Asia`, 보드라이프 `윙스팬 아시아`

## 🌍 세계관 / 테마

아시아에 사는 다양한 새들을 보호구역에 추가한다. 제품 자체는 1-2인 독립 게임으로도 작동하고, 기존 윙스팬에 섞는 카드 확장으로도 작동한다.

## 🏆 승리 조건

기본 최종 점수 구조는 윙스팬과 같다.

듀엣 모드에서는 추가로 듀엣 지도에서 가장 큰 연결 그룹 점수를 계산한다. 플록 모드는 개인전이며, 6-7명이 같은 라운드 목표를 공유해 경쟁한다.

## 🔁 전체 게임 흐름

아시아를 단순 카드 확장으로 쓰면 기본 흐름은 거의 변하지 않는다.

듀엣 모드에서는 `새를 놓는다 -> 해당 칸의 듀엣 토큰을 지도에 놓는다 -> 지도 보상을 받거나 라운드 목표를 준비한다 -> 게임 종료 때 연결 그룹 점수를 더한다`의 흐름이 추가된다.

플록 모드에서는 `두 그룹으로 나눈다 -> 각 그룹이 자기 새 트레이와 새 모이통을 쓴다 -> 두 명이 동시에 턴을 진행한다 -> 전체 테이블이 라운드 목표와 일부 상호작용을 공유한다`의 흐름이 된다.

## 준비 정보

### 주요 구성물

- 아시아 새 카드 90장
- 보너스 카드 14장
- 듀엣 지도/플록 목표판 양면 보드
- 듀엣 토큰 30개
- 듀엣 라운드 목표 타일 6개
- 플록 턴 순서 다이얼
- 새 먹이통 보드, 새 트레이, 먹이 주사위
- 2인용 플레이어 매트와 행동 큐브
- 오토마 구성물

### 셋업 요약: 카드 확장으로 사용할 때

1. 아시아 새 카드를 기존 새 덱에 섞는다.
2. 아시아 보너스 카드를 기존 보너스 카드에 섞는다.
3. 일반 윙스팬 규칙으로 진행한다.
4. 듀엣/플록 모드를 하지 않는다면 듀엣 지도와 플록 다이얼은 사용하지 않는다.

### 셋업 요약: 듀엣 모드

1. 2명이 아시아 구성물로 기본 셋업을 한다.
2. 일반 목표판 대신 듀엣 지도를 사용한다.
3. 빨간 모서리 표시가 있는 듀엣 라운드 목표 타일 4개를 무작위로 배치한다.
4. 각 플레이어는 자기 플레이어 매트의 15개 새 칸마다 듀엣 토큰 1개씩을 놓는다.
5. 나머지 시작 카드/음식/보너스 카드 선택은 기본판처럼 진행한다.

### 셋업 요약: 플록 모드

1. 기본판과 결합해 6명 또는 7명이 준비한다.
2. 플레이어를 두 그룹으로 나눈다. 6인은 3명+3명, 7인은 3명+4명이다.
3. 각 그룹은 별도의 새 트레이와 새 모이통을 사용한다.
4. 라운드 목표는 전체 테이블이 공유한다.
5. 턴 순서 다이얼을 놓고 시작 플레이어에 맞춘다.

## 🎮 턴 구조

기본 턴 구조는 같다.

1. 새 놓기
2. 먹이 얻기
3. 알 낳기
4. 새 카드 뽑기

듀엣 모드의 핵심 변화는 새를 놓을 때 듀엣 토큰을 지도에 놓는 것이다. 플록 모드의 핵심 변화는 두 명이 동시에 자기 그룹의 구성물로 턴을 진행하는 것이다.

## 🧱 페이즈별 설명

### 1. 듀엣 토큰 놓기

듀엣 모드에서 새를 놓으면, 방금 새를 놓은 칸에 있던 듀엣 토큰을 가져와 듀엣 지도에 놓는다.

지도 칸은 방금 놓은 새의 특징과 맞아야 한다. 예시는 다음과 같다.

- 새를 놓은 서식지
- 둥지 종류
- 음식 비용 아이콘
- 날개 길이 조건
- 부리 방향 조건

토큰을 보너스 아이콘이 있는 칸에 놓으면 즉시 그 보상을 받는다. 일부 칸은 나중에 새 먹이통이나 새 카드 트레이를 리셋하는 데 쓸 수 있다.

### 2. 듀엣 라운드 목표

듀엣 모드는 전용 라운드 목표를 사용한다. 라운드 종료 때 목표를 평가하고, 점수는 듀엣 목표 타일의 지시에 따른다.

듀엣 모드에서는 라운드 목표 결과에 따라 시작 플레이어 토큰 전달 방식이 달라진다. 목표에서 진 플레이어가 다음 라운드 시작 플레이어가 되고, 동률이면 상대에게 넘어간다.

### 3. 듀엣 게임 종료 점수

게임 종료 때 듀엣 지도에서 자신의 토큰 중 가장 큰 연결 그룹을 확인한다. 해당 그룹의 토큰 1개당 1점을 얻는다.

듀엣 토큰은 단순 보너스가 아니라 게임 종료 점수와 라운드 목표 모두에 영향을 준다.

### 4. 플록 모드 진행

플록 모드는 6-7인을 빠르게 진행하기 위해 두 명이 동시에 턴을 한다.

- 각 플레이어는 개인전으로 경쟁한다.
- 각 그룹은 자기 그룹의 새 트레이와 새 모이통만 사용한다.
- 음식 주사위를 참조하는 능력에서 `아무 주사위`라고 해도 자기 그룹의 주사위만 접근한다.
- 라운드 목표, 분홍색 능력, 다른 플레이어에게 영향을 주는 능력은 전체 테이블을 대상으로 공유한다.

### 5. 게임 종료 능력

아시아에는 노란색 `게임 종료` 능력이 있다. 4라운드 종료 절차 후, 최종 점수 계산 전에 처리한다. 여러 장이 있으면 원하는 순서로 처리한다.

게임 종료 능력은 분홍색 능력을 유발하지 않는다.

## ⚠️ 자주 헷갈리는 규칙 / 예외 상황

- 아시아는 카드 확장으로만 쓰면 듀엣/플록 규칙을 적용하지 않는다.
- 듀엣 토큰은 새를 놓을 때만 지도에 놓는다.
- 듀엣 토큰은 방금 놓은 새의 특징과 맞는 지도 칸에 놓아야 한다.
- 듀엣 지도 보상은 토큰을 놓을 때 즉시 받는다.
- 플록 모드에서 각 그룹은 자기 새 트레이와 새 모이통만 쓴다.
- 플록 모드에서도 점수 경쟁은 개인전이다.
- 노란색 게임 종료 능력은 최종 점수 계산 전에 처리한다.

## 자주 하는 실수

- 듀엣 모드에서 새를 놓고 듀엣 토큰을 지도에 놓지 않는다.
- 듀엣 토큰을 아무 지도 칸에 놓는다.
- 플록 모드에서 다른 그룹의 새 트레이나 주사위를 사용한다.
- 플록 모드의 두 그룹을 팀전으로 착각한다.
- 게임 종료 능력을 최종 점수 계산 후에 처리한다.

## 💡 초보자 팁

- 처음 2인 플레이는 듀엣 모드로 해도 좋지만, 듀엣 지도 조건을 천천히 확인한다.
- 듀엣 모드에서는 좋은 새를 놓는 것만큼 토큰 위치가 중요하다.
- 지도 보상 칸은 초반 자원 부족을 빠르게 풀어 준다.
- 플록 모드는 첫 플레이 시간이 길 수 있다. 모두가 기본 윙스팬 턴 구조를 알고 있을 때 추천한다.

## 모순/변경점

- Stonemaier는 제품을 1-7인으로 표기하지만, 보드라이프와 BGG 계열 기본 게임 정보는 단독 게임 기준 1-2인으로 표시한다. 6-7인은 기본판 결합 플록 모드다.
- 한국어 공식 룰북 PDF는 이번 검색에서 직접 확인하지 못했다. 국내판 용어는 실제 룰북과 다를 수 있음.

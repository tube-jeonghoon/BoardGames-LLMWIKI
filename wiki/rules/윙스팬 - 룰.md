# 🎲 윙스팬

## 🧩 한줄 요약

새를 숲, 초원, 습지에 배치하고 각 서식지의 먹이/알/카드 엔진을 굴려 4라운드 후 가장 높은 점수를 얻는 게임이다.

## 📌 기본 정보

| 항목 | 내용 |
| --- | --- |
| 최소인원 | 1명 |
| 최대인원 | 5명 |
| 추천인원 | BGG: 커뮤니티 추천 1-4인, 베스트 3인 / 보드라이프: 베스트 3인, 추천 2인 |
| 메커니즘 | BGG: Card Drafting, Dice Rolling, End Game Bonuses, Hand Management, Set Collection / 보조 분류: 엔진 빌딩, 태블로 빌딩 |
| 난이도 | BGG: 2.48/5 / 보드라이프: 2.48 |
| 플레이 시간 | BGG: 40-70분 / 보드라이프: 40-70분 / Stonemaier Games: 40-70분 |

> 확인 출처: BGG `Wingspan`, 보드라이프 `윙스팬`

출처별 차이: 추천 인원은 BGG가 1-4인 추천/3인 베스트, 보드라이프가 3인 베스트/2인 추천으로 표시한다. 이번 작성은 기본판 기준이며, 확장과 스팀/앱판의 자동화 처리는 제외했다.

## 🌍 세계관 / 테마

플레이어는 조류 애호가, 연구자, 보호구역 관리자처럼 각자의 조류 보호구역을 발전시킨다. 새는 숲, 초원, 습지 중 자신이 살 수 있는 서식지에 들어가고, 먹이를 먹고 알을 낳으며 다른 새와 조합을 만든다.

## 🏆 승리 조건

4라운드가 끝나면 최종 점수를 계산한다. 가장 높은 점수를 얻은 플레이어가 승리한다.

점수는 아래 항목을 합산한다.

- 새 카드에 적힌 점수
- 보너스 카드 점수
- 라운드 종료 목표 점수
- 새 위에 놓인 알
- 새 위에 저장한 먹이
- 새 밑에 끼워 넣은 카드

동점이면 사용하지 않은 먹이 토큰이 더 많은 플레이어가 승리한다. 그래도 같으면 공동 승리한다.

## 🔁 전체 게임 흐름

윙스팬은 `새를 놓는다 -> 서식지 행동으로 먹이/알/카드를 얻는다 -> 새 능력을 발동해 엔진을 키운다 -> 라운드 목표와 최종 점수를 얻는다`의 흐름이다.

초반에는 먹이와 카드가 부족해서 새를 많이 놓기 어렵다. 새가 늘어나면 각 서식지 행동이 강해지고, 행동 한 번으로 얻는 자원과 점수가 커진다. 하지만 라운드가 끝날 때마다 행동 큐브 1개가 목표판에 남기 때문에, 1라운드 8턴, 2라운드 7턴, 3라운드 6턴, 4라운드 5턴으로 점점 행동 수가 줄어든다.

## 준비 정보

### 주요 구성물

- 새 카드
- 보너스 카드
- 개인 보드
- 먹이 주사위와 새 모이통
- 먹이 토큰
- 알 토큰
- 행동 큐브
- 라운드 목표 타일과 목표판
- 점수 기록지
- 자동 플레이용 오토마 구성물

### 셋업 요약

1. 각 플레이어는 개인 보드와 행동 큐브 8개를 받는다.
2. 각 플레이어는 새 카드 5장과 먹이 5종을 1개씩 받는다.
3. 각 플레이어는 보너스 카드 2장을 받는다.
4. 시작 손패를 정한다. 남길 새 카드 1장마다 먹이 1개를 버리고, 보너스 카드는 1장만 남긴다.
5. 새 카드 덱에서 공개 카드 3장을 놓는다.
6. 먹이 주사위를 새 모이통에 굴려 넣는다.
7. 라운드 목표 타일을 목표판에 배치한다.
8. 시작 플레이어를 정한다.

## 🎮 턴 구조

자기 차례에는 행동 큐브 1개를 사용해 아래 네 행동 중 하나를 한다.

1. 새 놓기
2. 먹이 얻기
3. 알 낳기
4. 새 카드 뽑기

서식지 행동을 선택했다면 기본 행동을 처리한 뒤, 그 줄에 놓인 갈색 능력을 오른쪽에서 왼쪽으로 활성화한다. 모든 플레이어가 행동 큐브를 다 사용하면 라운드가 끝난다.

## 🧱 페이즈별 설명

### 1. 새 놓기

손패의 새 카드 1장을 골라 그 카드가 허용하는 서식지 중 하나에 놓는다.

1. 새 카드의 먹이 비용을 지불한다.
2. 놓을 서식지의 가장 왼쪽 빈 칸에 카드를 놓는다.
3. 카드가 들어가는 열에 따라 알 비용을 지불한다.
   - 1열: 알 비용 없음
   - 2-3열: 알 1개
   - 4-5열: 알 2개
4. 흰색 `새를 놓을 때` 능력이 있으면 처리한다.

새를 놓는 행동은 엔진을 만드는 행동이다. 새가 많아질수록 해당 서식지 행동이 강해지지만, 뒤쪽 열로 갈수록 알 비용이 필요하다.

### 2. 먹이 얻기

숲 서식지 행동이다. 행동 큐브를 숲 줄의 가장 왼쪽 공개 행동 칸에 놓고, 표시된 만큼 먹이 주사위를 선택해 먹이 토큰을 얻는다.

먹이를 얻은 뒤 숲 줄의 갈색 능력을 오른쪽에서 왼쪽으로 활성화한다. 일부 칸은 카드나 알을 지불해 추가 먹이를 얻는 선택지를 제공한다.

새 모이통이 비어 있거나 남은 주사위가 모두 같은 먹이 종류만 보여 줄 때는 룰북 조건에 따라 주사위를 다시 굴릴 수 있다. 주사위 한 면에 먹이가 2종 표시되어 있으면 그중 하나를 선택해 얻는다.

### 3. 알 낳기

초원 서식지 행동이다. 행동 큐브를 초원 줄의 가장 왼쪽 공개 행동 칸에 놓고, 표시된 수만큼 알을 낳는다.

알은 자신의 새 카드 위에 놓는다. 각 새 카드에는 둥지 수용량이 있으므로, 그 한도를 넘겨 알을 놓을 수 없다. 알을 놓은 뒤 초원 줄의 갈색 능력을 오른쪽에서 왼쪽으로 활성화한다.

알은 최종 점수이면서, 새를 뒤쪽 열에 놓기 위한 비용이기도 하다.

### 4. 새 카드 뽑기

습지 서식지 행동이다. 행동 큐브를 습지 줄의 가장 왼쪽 공개 행동 칸에 놓고, 표시된 수만큼 새 카드를 뽑는다.

카드는 공개된 3장 중에서 가져오거나 덱 맨 위에서 비공개로 뽑을 수 있다. 공개 카드 중 가져간 자리는 일반적으로 그 행동의 카드 선택이 끝난 뒤 보충한다. 카드를 뽑은 뒤 습지 줄의 갈색 능력을 오른쪽에서 왼쪽으로 활성화한다.

### 5. 새 능력

새 능력은 색으로 발동 시점이 나뉜다.

- 흰색: 새를 놓을 때 한 번 발동한다.
- 갈색: 해당 서식지 행동을 할 때 발동한다.
- 분홍색: 상대 턴에 조건이 맞으면 발동한다. 일반적으로 자기 차례가 돌아오기 전까지 한 번만 발동한다.

카드 문구가 기본 규칙보다 우선한다. 능력의 선택 여부, 비용, 대상, 순서는 카드 문구를 따른다.

### 6. 라운드 종료

모든 플레이어가 행동 큐브를 다 사용하면 라운드를 종료한다.

1. 이번 라운드의 목표를 평가한다.
2. 각 플레이어는 행동 큐브 1개를 목표판의 해당 위치에 놓는다.
3. 나머지 행동 큐브를 회수한다.
4. 공개 새 카드를 정리하고 새로 공개한다.
5. 시작 플레이어 마커를 다음 플레이어에게 넘긴다.

목표판에 놓은 큐브는 돌아오지 않기 때문에 다음 라운드의 행동 수가 1개 줄어든다.

### 7. 게임 종료와 최종 점수

4라운드 종료 후 최종 점수를 계산한다.

- 새 카드 점수
- 보너스 카드 점수
- 라운드 목표 점수
- 알 점수
- 저장한 먹이 점수
- 끼워 넣은 카드 점수

## ⚠️ 자주 헷갈리는 규칙 / 예외 상황

- 새를 놓을 때 먹이 비용만 보는 실수가 많다. 2열 이후에는 알 비용도 확인한다.
- 서식지 행동의 갈색 능력은 그 줄에 놓인 새들을 오른쪽에서 왼쪽으로 처리한다.
- 새 능력은 색에 따라 발동 시점이 다르다. 흰색은 놓을 때, 갈색은 서식지 행동 때, 분홍색은 상대 턴 조건 반응이다.
- 알은 점수이자 비용이다. 뒤쪽 열에 새를 많이 놓을 계획이면 알을 너무 일찍 소모하지 않는다.
- 저장한 먹이는 개인 공급의 먹이가 아니라 새 카드 위에 올라간 먹이다.
- 끼워 넣은 카드는 손패나 버림 카드가 아니라 새 카드 밑에 들어간 카드이며, 최종 점수에서 1장당 1점으로 계산한다.
- 공개 새 카드 보충 타이밍은 판본/디지털 구현 설명에서 헷갈릴 수 있다. 기본판 룰북 기준으로 카드 뽑기 행동의 선택이 끝난 뒤 보충하는 흐름으로 처리한다.
- 새 모이통 재굴림은 아무 때나 하는 것이 아니라, 비어 있거나 남은 주사위가 모두 같은 먹이 종류만 보여 주는 조건에서 가능하다.

## 자주 하는 실수

- 초반에 새만 많이 들고 먹이 생산을 키우지 못한다.
- 알이 최종 점수라서 전부 보존하려다가 새를 놓을 타이밍을 놓친다.
- 갈색 능력을 왼쪽에서 오른쪽으로 처리한다.
- 분홍색 능력을 상대 턴마다 여러 번 발동한다.
- 라운드 목표 점수 후 행동 큐브 1개를 목표판에 남기는 것을 잊는다.
- 최종 점수에서 저장한 먹이와 끼워 넣은 카드를 빼먹는다.

## 💡 초보자 팁

- 초반에는 숲, 초원, 습지 중 적어도 두 줄의 기본 생산을 빠르게 올린다.
- 먹이 비용이 낮고 능력이 자주 발동되는 새가 초반 엔진에 좋다.
- 보너스 카드 조건은 처음부터 전부 맞추려 하기보다, 이미 잘 돌아가는 서식지와 겹치는 조건을 노린다.
- 알은 점수지만 새 놓기 비용이기도 하므로 중반 이후 필요한 알 수를 미리 계산한다.
- 라운드 목표를 무리하게 쫓다가 엔진 성장이 멈추면 최종 점수에서 밀릴 수 있다.
- 카드 문구가 길면 색과 발동 시점부터 확인한다.

## 모순/변경점

- 추천 인원은 BGG와 보드라이프 표시가 다르다. BGG는 1-4인 추천/3인 베스트, 보드라이프는 3인 베스트/2인 추천으로 기록했다.
- 플레이 시간은 이번 확인 출처에서 BGG와 보드라이프가 40-70분으로 표시한다. Stonemaier 제품 설명도 40-70분 기준으로 반영했다.
- 한국어 공식 룰북 PDF 원문은 이번 검색에서 직접 확인하지 못했다. 국내판 카드명과 용어는 실제 룰북/카드와 다를 수 있음.

# 🎲 기즈모

## 🧩 한줄 요약

에너지 구슬을 모아 `기즈모를 만들고 -> 행동마다 카드 효과를 발동하고 -> 효과가 다시 행동을 부르는 연쇄 반응으로 점수를 만드는` 엔진 빌딩 게임이다.

## 📌 기본 정보

| 항목 | 내용 |
| --- | --- |
| 최소인원 | 2명 |
| 최대인원 | 4명 |
| 추천인원 | BGG: 베스트 3인, 2-4인 플레이 가능/추천 / 보드라이프: 베스트 3인, 추천 2인 |
| 메커니즘 | BGG: Action Queue, Card Drafting, Contracts, Open Drafting 등 / CMON: 에너지 구슬 수집, 기즈모 건설, 연쇄 반응형 엔진 빌딩 |
| 난이도 | BGG: 약 2.02/5 / 보드라이프: 2.05 |
| 플레이 시간 | CMON/BGG/보드라이프: 40-50분 |

> 확인 출처: BGG `Gizmos`, 보드라이프 `기즈모`

출처별 차이: 추천인원은 BGG와 보드라이프 모두 3인을 베스트로 보지만, 보드라이프는 추천 2인을 별도 표시한다. 난이도는 BGG 계열 자료가 약 2.02, 보드라이프가 2.05로 약간 다르다.

## 🌍 세계관 / 테마

플레이어는 거대한 과학 박람회에 참가한 발명가다. 네 종류의 에너지 구슬을 사용해 작은 장치인 기즈모를 만들고, 이 장치들이 서로 반응해 점점 더 큰 기계를 완성한다.

테마적으로는 “작은 기능이 연결될수록 거대한 연쇄 반응이 된다”는 발명/기계 조립 감각이 핵심이다. 구슬은 에너지 자원이고, 카드는 그 에너지를 소비해 붙이는 장치다.

## 🏆 승리 조건

게임 종료 후 가장 많은 승점을 가진 플레이어가 승리한다.

최종 점수는 아래를 더한다.

- 건설한 기즈모 카드에 인쇄된 승점
- 게임 중 얻은 승점 토큰

동점이면 아래 순서로 승자를 정한다.

1. 활성 기즈모 영역에 있는 기즈모가 더 많은 플레이어
2. 에너지 저장 링에 남은 에너지가 더 많은 플레이어
3. 시작 플레이어로부터 시계 방향으로 더 먼 플레이어

## 🔁 전체 게임 흐름

기즈모의 큰 흐름은 아래와 같다.

1. 에너지 줄에서 필요한 에너지 구슬을 모은다.
2. 공개 카드나 보관소 카드 중 하나를 건설한다.
3. 건설한 기즈모를 행동 종류별 영역에 놓는다.
4. 이후 같은 종류의 행동을 하면 해당 기즈모들이 발동한다.
5. 발동 효과가 추가 에너지, 추가 건설, 추가 연구, 승점 등을 만든다.
6. 연쇄 반응으로 한 턴의 효율을 키운다.
7. 4번째 레벨 3 기즈모 또는 총 16번째 기즈모가 나오면 종료를 준비한다.

핵심은 “지금 당장 점수가 높은 카드”보다 “내가 자주 할 행동에 반응하는 카드”를 연결하는 것이다. 같은 Pick 행동이라도 내 엔진이 잘 짜여 있으면 에너지 1개가 아니라 여러 보상으로 이어진다.

## 준비 정보

### 주요 구성물

- 에너지 분배기
- 에너지 구슬 4종
- 에너지 저장 링
- 플레이어 대시보드
- 시작 기즈모 카드
- 레벨 1/2/3 기즈모 카드
- 승점 토큰
- 효과 목록
- 룰북과 분배기 조립 가이드

### 셋업 요약

1. 에너지 분배기를 조립하고 모든 에너지 구슬을 넣는다.
2. 에너지 줄에 구슬 6개가 보이도록 채운다.
3. 레벨 1/2/3 기즈모 덱을 따로 섞는다.
4. 공개 카드 진열을 만든다. 보통 레벨 1은 4장, 레벨 2는 3장, 레벨 3은 2장을 공개한다.
5. 각 플레이어는 대시보드, 에너지 저장 링, 시작 기즈모를 받는다.
6. 시작 플레이어를 정한다.

각 플레이어의 시작 한계:

- 보관소 한도: 1장
- 에너지 저장 한도: 5개
- 연구량: 3장

이 수치는 업그레이드 기즈모로 늘어날 수 있다.

## 🎮 턴 구조

자기 턴에는 아래 네 행동 중 하나를 선택한다.

1. File
2. Pick
3. Build
4. Research

행동을 마친 뒤, 그 행동으로 발동한 기즈모 효과들을 원하는 순서로 처리한다. 효과로 추가 행동이 생기면, 그 추가 행동도 다시 다른 기즈모를 발동시킬 수 있다.

## 🧱 페이즈별 설명

### 1. File: 카드 보관

공개 진열의 기즈모 카드 1장을 골라 자기 보관소에 넣는다. 가져간 자리는 같은 레벨 덱에서 즉시 보충한다.

보관소 한도를 넘을 수 없다. 시작 한도는 1장이며, 업그레이드 기즈모를 통해 늘릴 수 있다.

File은 당장 건설하지 못하는 좋은 카드를 예약하는 행동이다. 다른 플레이어가 먼저 가져갈 위험을 줄일 수 있지만, 보관소 한도가 작기 때문에 너무 일찍 채우면 이후 선택지가 막힌다.

### 2. Pick: 에너지 가져오기

에너지 줄에 보이는 구슬 1개를 골라 자기 에너지 저장 링에 넣는다.

에너지 저장 한도를 넘을 수 없다. 시작 한도는 5개다. 저장 링이 가득 차면 새 에너지를 받을 수 없으며, 기존 에너지를 임의로 버리고 교체할 수 없다.

일부 효과는 에너지 줄에서 고르는 것이 아니라 분배기에서 무작위로 에너지를 뽑게 한다.

### 3. Build: 기즈모 건설

공개 진열 또는 자기 보관소의 기즈모 카드 1장을 선택하고, 카드 비용에 맞는 에너지를 지불해 건설한다.

건설한 기즈모는 플레이어 대시보드 아래의 해당 종류 영역에 놓는다. 같은 종류의 기즈모는 효과가 보이도록 겹쳐 둔다.

Converter 기즈모가 있다면 Build 중 에너지를 다른 색처럼 바꾸어 지불할 수 있다. Converter 세부 조건은 카드 효과를 따른다.

새로 건설한 기즈모는 같은 Build 행동으로는 발동하지 않는다. 하지만 그 턴 이후 다른 효과로 별도의 행동을 수행하면, 조건을 만족할 때 발동할 수 있다.

### 4. Research: 연구

레벨 1/2/3 덱 중 하나를 선택하고, 자기 연구량만큼 카드를 뽑아 본다. 시작 연구량은 3장이다.

뽑은 카드 중 1장을 선택해 아래 중 하나를 할 수 있다.

- 비용을 낼 수 있다면 즉시 Build
- 보관소에 공간이 있다면 File
- 아무것도 하지 않음

남은 카드는 선택한 레벨 덱 밑에 원하는 순서로 되돌린다.

Research 중 수행한 File 또는 Build도 실제 File/Build 행동으로 간주되어 관련 기즈모를 발동시킬 수 있다.

### 5. 기즈모 효과 발동

기즈모는 특정 행동에 반응해 효과를 낸다. 예를 들어 Pick에 반응하는 기즈모, Build에 반응하는 기즈모, File에 반응하는 기즈모 등이 있다.

처리 원칙:

- 행동을 완료한 뒤, 그 행동으로 발동한 기즈모들을 확인한다.
- 발동한 효과들의 실행 순서는 플레이어가 정한다.
- 효과로 새 행동을 수행하면 그 행동에 반응하는 다른 기즈모도 발동할 수 있다.
- 각 기즈모는 한 턴에 한 번만 사용할 수 있다.
- 같은 턴에 지은 기즈모도 이후 별도 행동으로 조건이 맞으면 사용할 수 있다.
- 단, 그 기즈모를 지은 바로 그 Build 행동으로는 발동하지 않는다.

### 6. 기즈모 종류 감각

기즈모는 크게 아래 성격으로 이해하면 쉽다.

- File 반응: 카드를 보관할 때 추가 이득을 준다.
- Pick 반응: 특정 색 에너지를 가져오거나 뽑을 때 추가 에너지/행동을 준다.
- Build 반응: 특정 방식으로 건설할 때 점수, 에너지, 추가 행동을 준다.
- Converter: Build 중 에너지 색을 바꾸거나 비용 지불을 유연하게 한다.
- Upgrade: 보관소 한도, 저장 한도, 연구량, 추가 보상 등을 늘린다.

첫 플레이에서는 카드 텍스트를 모두 외우기보다 “내가 자주 할 행동을 강화하는 카드인가?”를 기준으로 보면 된다.

### 7. 게임 종료

아래 중 하나가 발생하면 게임 종료가 촉발된다.

- 한 플레이어가 4번째 레벨 3 기즈모를 건설한다.
- 한 플레이어가 총 16번째 기즈모를 건설한다. 시작 기즈모 포함.

종료가 촉발되면 모든 플레이어가 같은 수의 턴을 갖도록 진행한다. 즉, 시작 플레이어의 오른쪽 플레이어가 턴을 마칠 때까지 진행한다.

### 8. 최종 점수

각 플레이어는 아래를 합산한다.

- 건설한 모든 기즈모 카드의 승점
- 보유한 승점 토큰

보관소에만 있고 건설하지 않은 카드는 점수가 되지 않는다.

## ⚠️ 자주 헷갈리는 규칙 / 예외 상황

- 자기 턴의 기본 행동은 1개뿐이다.
- 연쇄 반응으로 생긴 추가 행동도 실제 행동이므로 관련 기즈모를 발동시킬 수 있다.
- 각 기즈모는 한 턴에 한 번만 사용할 수 있다.
- 발동 효과 처리 순서는 플레이어가 정한다.
- 새로 지은 기즈모는 그 기즈모를 지은 Build 행동으로는 발동하지 않는다.
- Research는 한 레벨 덱에서만 뽑는다.
- Research 중 File/Build를 하면 File/Build 기즈모가 발동할 수 있다.
- 에너지 저장 링이 가득 차면 에너지를 더 받을 수 없다.
- 보관소 한도를 넘겨 File할 수 없다.
- 종료 조건의 16개 기즈모에는 시작 기즈모도 포함된다.
- 보관소에 있는 미건설 기즈모는 점수가 아니다.

## 자주 하는 실수

- Pick으로 에너지를 받을 때 저장 한도를 초과한다.
- Research에서 여러 레벨 덱을 섞어 뽑는다.
- 기즈모 하나를 같은 턴에 여러 번 사용한다.
- 새로 지은 기즈모를 그 Build 행동으로 바로 발동한다.
- 보관소에 있는 카드를 최종 점수에 포함한다.
- 연쇄 반응 중 발동 순서를 생각하지 않고 손해 보는 순서로 처리한다.
- 레벨 3 카드만 노리다가 초반 엔진이 늦어진다.
- 종료 조건 16개에 시작 기즈모를 빼고 계산한다.

## 💡 초보자 팁

- 초반에는 Pick이나 Build에 반응하는 낮은 레벨 기즈모를 먼저 깔아 엔진을 시작한다.
- 저장 한도 5개는 생각보다 작다. 필요한 색을 미리 정하고 에너지를 모은다.
- 보관소 한도 1장 때문에 File은 신중하게 한다. 정말 필요한 카드만 예약한다.
- Research는 공개 진열이 애매할 때 좋은 선택지다. 다만 뽑은 카드를 바로 못 지으면 보관소 공간도 필요하다.
- Converter는 점수가 낮아 보여도 고레벨 카드를 짓는 데 큰 도움이 된다.
- 한 행동에 너무 많은 카드가 몰리면 특정 행동만 강해진다. Pick, Build, Research 중 자주 쓸 행동을 중심으로 균형을 잡는다.
- 상대가 곧 4번째 레벨 3 또는 16번째 기즈모에 도달할 것 같으면, 내 마지막 턴 점수화를 준비한다.

## 모순/변경점

- 이 문서는 기본판 룰 기준이다. 프로모 세트와 팬메이드 솔로 변형은 제외했다.
- 1판과 2판은 에너지 분배기 재질 등 구성물 차이가 있을 수 있다. 핵심 룰은 동일한 것으로 정리했다.
- 추천인원은 BGG와 보드라이프 모두 3인을 베스트로 보지만, 보드라이프는 추천 2인을 별도 표시한다.
- 난이도는 BGG 계열 자료가 약 2.02, 보드라이프가 2.05로 약간 다르다.

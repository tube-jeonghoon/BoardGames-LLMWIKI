# 🎲 윙스팬: 오세아니아 확장

## 🧩 한줄 요약

새 플레이어 매트와 넥타르 음식을 추가해 먹이 운영을 유연하게 만들고, 서식지별 넥타르 다수 점수로 게임 종료 점수를 확장한다.

## 🌍 세계관 / 테마

호주와 뉴질랜드를 중심으로 한 오세아니아 조류가 추가된다. 꿀과 당분이 풍부한 생태계를 반영해 넥타르라는 새 음식 타입이 들어온다.

## 🏆 승리 조건

기본판 최종 점수에 서식지별 사용 넥타르 다수 점수가 추가된다.

각 서식지마다 사용 넥타르가 가장 많은 플레이어는 5점, 두 번째로 많은 플레이어는 2점을 받는다. 해당 서식지에 사용 넥타르가 1개 이상 있어야 점수를 받을 수 있다. 동률이면 해당 순위 점수를 합쳐 나누고 내림 처리한다.

## 🔁 전체 게임 흐름

기본 흐름은 [[윙스팬 - 룰]]과 같지만, 오세아니아 매트와 넥타르 때문에 자원 운영 감각이 달라진다.

`넥타르를 얻는다 -> 새 비용이나 행동 강화에 쓴다 -> 사용한 서식지 칸에 넥타르를 기록한다 -> 라운드 끝에 미사용 넥타르를 버린다 -> 게임 종료 때 서식지별 다수 점수를 받는다`의 흐름이다.

넥타르는 강력하지만 라운드를 넘겨 들고 갈 수 없으므로, 얻은 라운드 안에서 쓰는 계획이 중요하다.

## 준비 정보

### 주요 구성물

- 오세아니아 새 카드 95장
- 새 플레이어 매트 5장
- 넥타르 토큰
- 넥타르가 포함된 새 먹이 주사위
- 보너스 카드 5장
- 라운드 목표 타일 4개
- 노란색 알
- 참조 타일
- 오토마 구성물

### 셋업 요약

1. 오세아니아 새 카드를 기존 새 덱에 섞는다.
2. 오세아니아 보너스 카드와 라운드 목표 타일을 기존 구성물에 섞는다.
3. 기본판 플레이어 매트 대신 오세아니아 새 매트를 사용한다.
4. 기본판 먹이 주사위 대신 오세아니아 먹이 주사위를 사용한다.
5. 넥타르 토큰을 공급처에 추가한다.
6. 시작 카드/음식 선택은 기본판처럼 진행하되, 선택 후 각 플레이어가 넥타르 1개를 받는 규칙을 사용한다.

## 🎮 턴 구조

자기 차례 행동은 기본판과 같다.

1. 새 놓기
2. 먹이 얻기
3. 알 낳기
4. 새 카드 뽑기

변화는 넥타르 지불, 오세아니아 매트의 행동 강화, 라운드 종료의 넥타르 정리, 게임 종료 능력에서 생긴다.

## 🧱 페이즈별 설명

### 1. 넥타르 얻기

새 먹이통에서 넥타르 면이 나온 주사위를 선택하면 넥타르 토큰을 얻는다. 넥타르는 개인 공급에 놓는다.

### 2. 넥타르 사용

넥타르는 아래 상황에서 와일드 음식처럼 쓸 수 있다.

- 새를 놓을 때 음식 비용 지불
- `아무 음식` 비용 지불
- 플레이어 매트의 행동 강화 비용 지불
- 카드 능력이 아무 음식 지불을 요구하는 경우

넥타르를 사용했다면 그 넥타르를 사용한 서식지의 `사용 넥타르` 칸에 놓는다. 새 놓기라면 그 새를 놓은 서식지에 놓고, 서식지 행동 강화라면 그 행동을 한 줄에 놓는다.

### 3. 넥타르가 와일드가 아닌 경우

카드 능력이 특정 음식 타입을 참조할 때 넥타르는 그 음식으로 세지 않는다. 예를 들어 새 먹이통에 쥐가 있는지 확인하는 능력에서 넥타르를 쥐로 보지 않는다.

또한 넥타르는 이미 와일드 음식이므로 2:1 변환의 재료로 버릴 수 없다. 반대로 다른 음식 2개를 써서 넥타르 비용 하나를 대신하는 것은 일반 2:1 규칙을 따른다.

### 4. 라운드 종료

오세아니아를 사용할 때 라운드 종료 순서는 아래처럼 기억한다.

1. 유럽 확장을 함께 쓰면 라운드 종료 능력을 처리한다.
2. 개인 공급에 남은 미사용 넥타르를 버린다.
3. 라운드 목표를 점수화한다.
4. 행동 큐브를 회수한다.
5. 새 카드 트레이를 정리하고 보충한다.
6. 4라운드가 끝났다면 게임 종료 능력을 처리하고 최종 점수를 계산한다. 아니라면 시작 플레이어 토큰을 넘긴다.

라운드 종료 때 버리는 것은 개인 공급의 미사용 넥타르뿐이다. 사용 넥타르 칸의 넥타르와 새 카드 위에 저장된 넥타르는 버리지 않는다.

### 5. 게임 종료 능력

노란색 `게임 종료` 능력은 4라운드 종료 절차를 마친 뒤 최종 점수 계산 전에 처리한다. 여러 장이 있으면 원하는 순서로 처리한다.

게임 종료 능력은 분홍색 능력을 유발하지 않는다.

### 6. 최종 점수

기본 최종 점수에 각 서식지의 사용 넥타르 다수 점수를 더한다.

- 서식지별 1등: 5점
- 서식지별 2등: 2점
- 1개 이상 사용 넥타르가 있어야 점수 가능
- 동률은 관련 순위 점수를 합산해 나누고 내림

## ⚠️ 자주 헷갈리는 규칙 / 예외 상황

- 라운드가 끝나면 개인 공급의 미사용 넥타르는 버린다.
- 사용 넥타르 칸의 넥타르는 버리지 않는다.
- 새 위에 저장된 넥타르도 라운드 종료 때 버리지 않는다.
- 넥타르는 특정 음식 타입을 세는 능력에서는 와일드가 아니다.
- 넥타르는 2:1 변환의 재료로 버릴 수 없다.
- 사용 넥타르는 반드시 사용한 서식지 줄에 기록한다.
- 게임 종료 능력은 4라운드 종료 절차 후, 최종 점수 계산 전에 처리한다.

## 자주 하는 실수

- 넥타르를 라운드 사이에 계속 들고 간다.
- 넥타르를 아무 특정 음식 참조에도 모두 와일드로 계산한다.
- 사용 넥타르를 한 칸에 모아 놓고 서식지별 점수를 놓친다.
- 새 놓기에 쓴 넥타르를 새가 놓인 서식지가 아니라 다른 줄에 기록한다.
- 게임 종료 능력을 최종 점수 계산 뒤에 처리한다.

## 💡 초보자 팁

- 넥타르는 강력하지만 남기면 버려진다. 얻은 라운드 안에 쓸 계획을 세운다.
- 넥타르 점수는 3개 서식지로 나뉘므로, 한 줄에만 몰아 쓰는 것이 항상 최선은 아니다.
- 오세아니아 매트는 기본판보다 자원 흐름이 달라진다. 처음에는 매트 아이콘을 천천히 확인한다.
- 넥타르 다수 점수는 막판에 뒤집힐 수 있으니 4라운드에 상대의 사용 넥타르 칸을 확인한다.

## 모순/변경점

- 보드라이프는 베스트 3인/추천 2인으로 표시한다. BGG 추천 인원 상세값은 이번 검색에서 원문 페이지를 직접 열람하지 못했고, 보조 자료 기준으로만 확인했다.
- 한국어 공식 룰북 PDF는 이번 검색에서 직접 확인하지 못했다. 국내판 용어는 실제 룰북과 다를 수 있음.

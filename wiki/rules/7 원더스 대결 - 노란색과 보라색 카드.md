# 7 원더스 대결 - 노란색과 보라색 카드

## 한줄 요약

노란색 카드는 돈과 자원 구매 효율을 책임지는 상업 카드이고, 보라색 카드는 시대 III에 3장만 등장해 특정 조건으로 큰 점수를 주는 길드 카드다.

## 노란색 카드 기본 규칙

- 노란색 카드는 `상업 건물`이다.
- 효과는 크게 코인 획득, 자원 생산, 거래 비용 변경, 조건부 승점으로 나뉜다.
- 카드를 버리고 돈을 받을 때는 `2코인 + 자기 도시의 노란색 카드 수`를 받는다.
- 일부 노란색 카드가 생산하는 자원은 상대의 거래 비용을 올리지 않는다.
- 거래 할인 노란색 카드는 지정 자원을 은행에서 1코인으로 사게 한다. 상대가 해당 자원을 많이 생산해도 1코인이다.

| Lighthouse          | III | 건설 즉시 자기 노란색 카드마다 코인 1개를 받는다. 이 카드 자신도 센다. 게임 종료 때 자기 노란색 카드마다 승점 1점을 얻는다. |
| Armory              | III | 건설 즉시 자기 빨간색 카드마다 코인 1개를 받는다. 게임 종료 때 자기 빨간색 카드마다 승점 1점을 얻는다.              |

## 노란색 카드 운영 포인트

- 초반 `Reserve` 계열은 상대가 특정 원자재를 독점했을 때 비용 폭등을 막는다.
- `Customs House`는 과학/파란 카드 비용에 자주 필요한 유리/파피루스를 싸게 만든다.
- `Forum`과 `Caravansery`는 선택형 생산이지만, 한 턴에 표시된 자원 중 하나만 생산한다.
- 시대 III 노란 카드는 지은 순간 코인을 한 번 받고, 승점은 게임 종료 때 다시 계산한다. 이후 조건 카드가 늘어나면 승점은 늘 수 있지만, 이미 받은 코인은 다시 받지 않는다.

## 보라색 카드 기본 규칙

- 보라색 카드는 `길드`다.
- 시대 III 준비 때 길드 카드 7장 중 3장만 무작위로 시대 III 덱에 섞는다.
- 길드는 대체로 `건설 즉시 코인`과 `게임 종료 점수`를 준다.
- 코인은 건설할 때 한 번만 받는다.
- 게임 종료 점수를 계산할 때 선택하는 도시는, 건설 당시 코인을 받을 때 선택한 도시와 달라도 된다.

| 카드 | 효과 |

| Moneylenders Guild | 게임 종료 때, 가장 부유한 도시의 3코인 세트마다 1점. |

| Shipowners Guild | 건설 즉시 갈색+회색 카드가 더 많은 도시의 갈색/회색 카드마다 코인 1개. 게임 종료 때 같은 방식으로 카드마다 1점. 갈색과 회색은 한 도시를 합쳐 센다. |

| Magistrates Guild | 건설 즉시 파란색 카드가 더 많은 도시의 파란색 카드마다 코인 1개. 게임 종료 때 파란색 카드가 더 많은 도시의 파란색 카드마다 1점. |
| Tacticians Guild | 건설 즉시 빨간색 카드가 더 많은 도시의 빨간색 카드마다 코인 1개. 게임 종료 때 빨간색 카드가 더 많은 도시의 빨간색 카드마다 1점. |

## 자주 헷갈리는 점

- 길드는 내 도시만 세는 카드가 아니다. 카드마다 `두 도시 중 조건이 더 큰 도시`를 본다.
- 길드로 받은 코인은 건설 순간 한 번만 받는다. 나중에 조건 카드가 늘어도 코인을 추가로 받지 않는다.
- 길드의 게임 종료 점수는 종료 시점 기준으로 다시 계산한다.
- 노란색 카드의 거래 할인은 상대 생산량을 무시하고 지정 자원 비용을 1코인으로 고정한다.
- 노란색 카드/불가사의가 생산하는 자원은 상대가 그 자원을 살 때 비용 증가에 포함하지 않는다.

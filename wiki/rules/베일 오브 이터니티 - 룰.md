# 🎲 베일 오브 이터니티

## 🧩 한줄 요약

공개된 생명체를 사냥하고, 팔아서 마법석을 얻거나 손패로 길들인 뒤, 비용을 내고 소환해 카드 효과 엔진으로 60점을 노리는 카드 드래프팅/엔진 빌딩 게임이다.

## 📌 기본 정보

| 항목     | 내용                                                                                                   |
| ------ | ---------------------------------------------------------------------------------------------------- |
| 최소인원   | 2명                                                                                                   |
| 최대인원   | 4명                                                                                                   |
| 추천인원   | BGG: 추천 2-4인, 베스트 3인 / 보드라이프: 베스트 3인, 추천 2인                                                          |
| 메커니즘   | BGG/보조 정보: Hand Management, Market, Open Drafting, Set Collection / 보드라이프: 상세 진행방식은 이번 검색에서 직접 확인 못함 |
| 난이도    | BGG: 약 2.15/5 / 보드라이프: 2.15                                                                          |
| 플레이 시간 | BGG/보드라이프: 30-45분 / Renegade: 40분                                                                    |
| 평점     | BGG Rating: 7.2/10 / 보드라이프: 7.5/10                                                                   |

> 확인 출처: BGG `The Vale of Eternity`, 보드라이프 `베일 오브 이터니티`

출처별 차이: BGG는 추천 2-4인/베스트 3인으로, 보드라이프는 베스트 3인/추천 2인으로 표시한다. 플레이 시간은 BGG와 보드라이프가 30-45분, Renegade 공식 페이지가 40분으로 표시한다.

## 🌍 세계관 / 테마

플레이어는 신화 속 생명체를 사냥하고 길들이는 조련사다. 불, 물, 대지, 바람, 용 계열의 생명체를 소환해 능력을 활용하고, 가장 뛰어난 생명체 조합을 만든다.

## 🏆 승리 조건

라운드 종료 시 누군가 60점 이상이거나 10라운드가 끝나면 게임이 끝난다. 가장 높은 점수의 플레이어가 승리한다.

동점이면 소환된 카드가 더 많은 플레이어가 승리한다. 그래도 같으면 공동 승리한다.

## 🔁 전체 게임 흐름

베일 오브 이터니티는 `카드를 고른다 -> 팔아서 마법석을 얻거나 손패로 길들인다 -> 마법석으로 카드를 소환한다 -> 소환된 카드 효과로 점수와 자원을 만든다 -> 60점을 노린다`의 흐름이다.

핵심은 카드와 마법석, 영역 공간의 균형이다. 좋은 카드를 많이 가져와도 마법석이 없으면 소환할 수 없고, 마법석이 많아도 손패에 점수 카드가 없으면 점수가 나지 않는다. 또 소환된 카드 수는 현재 라운드 수를 넘을 수 없으므로, 언제 소환하고 언제 제거할지도 중요하다.

## 준비 정보

### 주요 구성물

- 게임 보드와 이터니티 스탠드
- 점수 보드
- 마법석 42개
  - 빨간 마법석 16개: 가치 1
  - 파란 마법석 14개: 가치 3
  - 보라 마법석 12개: 가치 6
- 시작 플레이어 마커 1개
- 라운드 마커 1개
- 플레이어 마커 8개
- 점수 마커 4개
- 생명체 카드 70장
- 플레이어 타일 4개

### 셋업 요약

1. 각 플레이어는 색 하나를 고르고 플레이어 마커 2개와 플레이어 타일 1개를 받는다.
2. 최근에 파충류를 본 플레이어가 시작 플레이어가 된다.
3. 게임 보드와 점수 보드를 가운데 놓는다.
4. 모든 카드를 섞어 더미를 만들고, 버린 카드 더미 공간을 둔다.
5. 라운드 마커를 1라운드 칸에 놓는다.
6. 시작 플레이어부터 시계 방향 순서에 따라 순서값만큼 시작 점수를 받는다.
7. 마법석을 공급처에 둔다.

## 🎮 턴 구조

게임은 여러 라운드로 진행된다. 각 라운드는 아래 3단계로 진행된다.

1. 사냥 단계
2. 액션 단계
3. 해결 단계

라운드 종료 후 종료 조건을 확인한다. 끝나지 않았다면 라운드 마커를 전진시키고, 시작 플레이어 마커를 왼쪽 플레이어에게 넘긴다.

## 🧱 페이즈별 설명

### 1. 사냥 단계

플레이어 수의 2배만큼 카드를 공개한다. 공개한 카드는 계열별로 게임 보드 주변 해당 영역에 놓는다. 카드 더미가 떨어지면 버린 카드 더미를 섞어 새 더미를 만든다.

카드 선택은 두 번에 나누어 진행한다.

1. 시작 플레이어부터 시계 방향으로 각자 카드 1장에 자기 마커를 놓는다.
2. 마지막 플레이어부터 반시계 방향으로 각자 두 번째 카드에 남은 마커를 놓는다.

이미 다른 플레이어 마커가 놓인 카드는 선택할 수 없다. 마지막 플레이어는 첫 번째 선택과 두 번째 선택을 연속으로 하게 된다.

### 2. 액션 단계

시작 플레이어부터 시계 방향으로 차례를 가진다. 자기 차례에는 원하는 만큼 액션을 원하는 순서로 수행할 수 있다. 같은 액션을 여러 번 해도 된다.

액션은 4가지다.

### 판매

사냥 단계에서 자기 마커를 놓은 카드 1장을 버리고, 해당 계열에 따른 마법석을 받는다. 이때 카드의 소환 비용은 보지 않는다.

판매 보상:

| 계열 | 보상 |
| --- | --- |
| 불 | 빨간 마법석 3개 |
| 물 | 파란 마법석 1개 |
| 대지 | 빨간 마법석 4개 |
| 바람 | 빨간 마법석 1개 + 파란 마법석 1개 |
| 용 | 보라 마법석 1개 |

마법석은 종류와 무관하게 최대 4개까지만 보유할 수 있다. 초과하면 4개만 남기고 나머지는 버린다.

### 길들이기

사냥 단계에서 자기 마커를 놓은 카드 1장을 손패로 가져온다. 손패 제한은 없다.

### 소환

손패의 카드 1장을 자신의 영역에 앞면으로 놓는다. 카드의 소환 비용만큼 마법석을 지불한다.

- 비용보다 많이 지불해도 거스름돈을 받지 않는다.
- 카드 효과로 비용이 줄어들 수 있으며, 여러 할인 효과는 누적될 수 있다.
- 비용은 0보다 낮아지지 않는다.
- 소환한 카드에 즉시 효과가 있으면 바로 해결한다.
- 자신의 영역이 가득 차 있으면 소환할 수 없다.

영역 수는 현재 라운드 수와 같다. 예를 들어 5라운드에는 소환된 카드를 최대 5장까지 둘 수 있다.

### 제거

자기 영역에 소환된 카드 1장을 버린다. 제거 비용은 현재 라운드 수와 같다.

예를 들어 6라운드에 카드를 제거하려면 가치 6 이상의 마법석을 지불해야 한다. 제거는 자기 카드에만 할 수 있다.

자신의 마커가 놓인 사냥 카드가 남아 있으면 차례를 끝낼 수 없다. 반드시 판매하거나 길들여 마커를 회수해야 한다.

### 3. 해결 단계

시작 플레이어부터 시계 방향으로, 자신의 영역에 있는 활성 효과를 해결한다.

- 각 활성 효과는 라운드마다 1번씩 사용할 수 있다.
- 여러 활성 효과가 있다면 원하는 순서로 해결한다.
- 사용할 수 있는 활성 효과가 없거나 모두 사용했다면 차례를 끝낸다.
- 카드 효과가 일반 규칙과 충돌하면 카드 효과가 우선한다.

## ⚠️ 자주 헷갈리는 규칙 / 예외 상황

- 마법석은 최대 4개까지만 보유한다. 가치가 큰 보라 마법석도 개수 1개로 센다.
- 마법석을 초과 보유하게 되면 다른 행동을 하기 전에 4개만 남기고 버린다.
- 마법석으로 비용을 초과 지불해도 거스름돈은 없다.
- 소환 가능한 카드 수는 현재 라운드 수와 같다.
- 손패 제한은 없다.
- 판매 보상은 카드 비용이 아니라 카드 계열로 결정된다.
- 사냥 단계에서 고른 카드는 액션 단계 중 반드시 판매하거나 길들여야 한다.
- 즉시 효과는 소환할 때만 발동한다.
- 지속 효과는 소환 후 계속 적용된다.
- 활성 효과는 해결 단계에 라운드당 1번 사용할 수 있다.
- 카드 효과가 규칙을 깨면 카드 효과가 우선한다.

## 자주 하는 실수

- 좋은 카드를 모두 길들이기만 하고 마법석을 확보하지 않는다.
- 마법석 4개 제한을 잊고 너무 많이 받는다.
- 6마법석으로 4비용 카드를 소환하고 차액 2를 받으려 한다. 거스름돈은 없다.
- 현재 라운드 수보다 많은 카드를 소환하려 한다.
- 사냥 카드에 자기 마커를 남긴 채 차례를 끝내려 한다.
- 활성 효과를 액션 단계에 사용한다. 활성 효과는 해결 단계에 사용한다.
- 종료 조건을 즉시 확인한다고 착각한다. 60점 이상 여부는 라운드 종료 시 확인한다.

## 💡 초보자 팁

- 초반에는 마법석을 만들어 주는 카드와 안정적인 점수 카드를 우선 본다.
- 매 라운드 두 카드 중 하나는 팔아 자원을 만들고, 하나는 길들여 다음 계획을 준비하는 흐름이 기본이다.
- 용 카드는 강하지만 비용이 높다. 무작정 들고 있기보다 소환할 마법석 계획이 있는지 확인한다.
- 영역 수는 라운드가 갈수록 늘어난다. 초반에 약한 카드를 너무 많이 소환하면 나중에 제거 비용을 내야 할 수 있다.
- 60점 도달 계획을 중반부터 세운다. 자원 생산만 하고 점수 카드가 없으면 게임이 끝날 때 따라잡기 어렵다.

## 모순/변경점

- 한국어 공식 룰북 PDF는 이번 검색에서 확인하지 못했다.
- BGG는 추천 2-4인/베스트 3인, 보드라이프는 베스트 3인/추천 2인으로 표기한다.
- 플레이 시간은 BGG/보드라이프 30-45분, Renegade 공식 페이지 40분으로 표기 차이가 있다.
- 본 페이지는 기본판 기준이다. `Artifacts`, `Curse` 확장 규칙은 포함하지 않았다.

# 그랜드 오스트리아 호텔 룰 검색 요약

## 요약

그랜드 오스트리아 호텔은 빈의 호텔을 운영하며 손님을 데려오고, 음식/음료 주문을 채우고, 준비한 객실에 투숙시켜 점수와 보너스 액션을 얻는 주사위 액션 드래프팅 게임이다. 게임은 7라운드로 진행되며, 각 라운드마다 주사위 눈금별 액션 강도가 달라진다.

## 기본 정보

- 최소인원: 2명
- 최대인원: 4명
- 추천인원:
  - BGG: 커뮤니티 추천 2-3인, 베스트 2인
  - 보드라이프: 현재 확인한 페이지에는 베스트/추천 인원 값 별도 표시 없음
- 플레이 시간:
  - BGG/보드라이프: 60-120분
  - Lookout Spiele: 약 90분
- 난이도:
  - BGG: 3.20/5
  - 보드라이프: 2.75/5
- 메커니즘:
  - BGG: Action Drafting, Contracts, Dice Rolling, End Game Bonuses, Hand Management, Set Collection, Turn Order: Progressive, Variable Set-up 등
  - Yucata: Dice game, Hand management, Set collection, Variable player powers

## 룰 핵심

- 7라운드 동안 진행한다.
- 3/5/7라운드 종료 시 황제 점수 계산을 한다.
- 각 라운드마다 시작 플레이어가 주사위를 굴려 1-6 액션 칸에 분류한다.
- 한 라운드에 각 플레이어는 2턴을 가진다.
- 턴마다 손님 데려오기는 선택, 주사위 선택과 액션 수행은 필수다.
- 주사위 액션의 강도는 해당 액션 칸에 남아 있던 주사위 수로 정한다.
- 음식/음료를 손님 주문에 놓고, 주문을 완성한 손님을 색이 맞는 준비된 객실로 보내 점수와 보상을 받는다.
- 객실 그룹을 모두 투숙 완료하면 색상별 객실 보너스를 얻는다.
- 정치 카드 조건을 달성하면 자기 턴 중 추가 행동으로 점수를 받을 수 있다.
- 최종 점수는 종료 직원 카드, 투숙 완료 객실, 남은 크로네/음식/음료, 카페에 남은 손님 페널티를 계산한다.

## 출처별 차이

- 추천인원: BGG는 베스트 2인/추천 2-3인으로 표시한다. 보드라이프 현재 페이지는 인원 2-4명은 확인되지만 베스트/추천 인원 값은 별도로 보이지 않는다.
- 난이도: BGG는 3.20/5, 보드라이프는 2.75/5로 차이가 크다.
- 플레이 시간: BGG/보드라이프는 60-120분, Lookout Spiele 공식 페이지는 약 90분으로 표시한다.

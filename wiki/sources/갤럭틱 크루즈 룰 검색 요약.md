# 갤럭틱 크루즈 룰 검색 요약

## 핵심 내용

- [[갤럭틱 크루즈]]는 1-4인 헤비 유로/일꾼 배치 게임이다. 우주 크루즈 회사를 운영하며 선박을 건조하고, 크루즈를 예약/광고하고, 승객을 태워 발사해 VP를 얻는다.
- 턴은 `선박 진행 -> 일꾼 배치/선박 발사/회의 중 하나 -> 회사 목표 달성 확인` 순서다.
- 일꾼 배치 액션은 네트워크 6개 장소에 놓인 12개 액션 타일과 개발 연결에 의해 결정된다. 배치한 일꾼은 접근 가능한 액션을 최대 2개 수행한다.
- 진행 트랙은 선박 발사와 회사 목표 달성으로 채워지며, 섹션 1/2는 중간 AGM 점수, 섹션 3은 게임 종료를 촉발한다.
- 최종 점수는 남은 보급품, 진행 점수, 평판, 선박 세그먼트, 개발, 조종석 점수 등을 합산한다.

## 새로 추가된 지식

- 게임 길이는 고정 3라운드가 아니라 진행 트랙이 차는 방식의 가변 라운드 구조다.
- 다른 플레이어의 일꾼을 밀어내면 밀려난 플레이어가 즉시 펀딩 보너스를 얻는다.
- 자기 차례 시작 때 우주에 있는 선박은 자동으로 다음 정류장으로 이동하며, 정류장 종류에 따라 목적지 점수/우주일 보너스/귀환 처리를 한다.
- 발사는 휴게실의 일꾼을 발사 엘리베이터에 놓아 5단계 카운트다운을 해결하는 별도 턴 선택지다.
- BGG와 보드라이프의 연도/추천인원/난이도/평점 값에 차이가 있어 기본 정보 검증 섹션에 출처별로 나눠 적었다.

## 모순/주의점

- Kinson Key Games 공식 페이지는 게임 개요를 제공하지만, 이 작업 시점에는 제작사 사이트에서 룰북 PDF 직접 링크를 확인하지 못했다. 세부 규칙은 공개 PDF 미러의 `Training Manual v1.2.5`를 내려받아 확인했다.
- 보드라이프는 2024년, BGG는 2025년으로 표기한다. 펀딩/출시/유통 기준 차이일 수 있어 단정하지 않는다.
- 보드라이프 난이도 3.92와 BGG 복잡도 4.00은 현재 검색 결과 기준이다. BGG 스니펫/캐시 값이 변동될 수 있다.

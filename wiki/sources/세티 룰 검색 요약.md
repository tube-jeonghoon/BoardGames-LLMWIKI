## 요약

세티는 5라운드 동안 우주 탐사 기관을 운영하며 탐사선, 스캔, 데이터 분석, 기술 연구, 카드 효과를 통해 외계 생명체의 흔적을 찾는 무거운 유로게임이다. 각 턴에는 주 행동 1개만 수행하지만, 자유 행동은 여러 번 수행할 수 있어 자원 전환, 탐사선 이동, 데이터 배치, 미션 완료가 턴 효율을 좌우한다.

핵심 흐름은 `자원을 모은다 -> 탐사선을 보내거나 별을 스캔한다 -> 데이터와 생명체 흔적을 얻는다 -> 기술과 카드로 엔진을 강화한다 -> 외계 종족을 발견한다 -> 5라운드 후 금색 점수 타일과 종료 점수 카드를 계산한다`이다.

## 핵심 확인 사항

- CGE/BGG/보드라이프 모두 1-4명.
- 플레이 시간은 BGG/보드라이프 40-160분, CGE는 플레이어당 40분.
- 추천 인원은 BGG 1-4인 추천/3인 베스트, 보드라이프 3인 베스트/2인 추천.
- 난이도는 BGG 3.82, 보드라이프 3.79.
- 게임은 5라운드.
- 주 행동은 탐사선 발사, 궤도 진입, 착륙, 스캔, 데이터 분석, 카드 사용, 기술 연구, 패스.
- 패스 후에는 해당 라운드에서 더 이상 턴을 하지 않고, 모든 플레이어가 패스하면 수입을 받는다.
- 최종 점수는 종료 점수 카드, 금색 점수 타일, 일부 외계 종족 추가 점수를 더한다.

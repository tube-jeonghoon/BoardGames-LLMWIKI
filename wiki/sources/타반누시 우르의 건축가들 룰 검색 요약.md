## 요약

타반누시: 우르의 건축가들은 고대 메소포타미아의 우르를 배경으로, 5개 구역을 오가며 주사위를 자원과 다음 이동지로 동시에 사용하는 헤비 유로 게임이다. 각 구역의 주사위가 비면 해당 구역 점수 계산이 즉시 일어나므로, 액션 효율뿐 아니라 어느 구역의 점수를 언제 터뜨릴지가 핵심이다.

핵심 흐름은 `현재 구역에서 주사위 선택 -> 주사위 숫자로 건축가 이동 -> 현재 구역 행동 2회 -> 보조자 이동 -> 주사위가 비면 구역 점수`이다. 일반 구역은 프로젝트와 건물, 정원/물로 점수 기반을 만들고, 항구와 지구라트는 배 능력과 별도 점수 조건을 제공한다.

## 핵심 확인 사항

- Board&Dice/BGG/보드라이프 모두 1-4명, 14세 이상, 2021년 게임으로 확인된다.
- 플레이 시간은 Board&Dice가 90-120분/120+분, BGG와 보드라이프가 120분으로 표시한다.
- BGG 추천 인원은 베스트 3-4인, 보드라이프는 베스트 4인/추천 2인으로 다르다.
- 난이도는 BGG/보드라이프 3.92/5, Board Game Oracle 3.9/5 수준이다.
- 평점은 BGG 계열 약 7.3-7.4/10, 보드라이프 6.5-6.6/10으로 차이가 있다.
- 언어의존도는 BGG `No necessary in-game text`, 보드라이프 `없음`으로 일치한다.
- 구역 주사위가 비면 중간 점수 계산이 발생하며, 게임 종료 후 모든 구역과 도시 카드를 최종 점수화한다.

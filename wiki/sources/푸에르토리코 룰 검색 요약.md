# 푸에르토리코 룰 검색 요약

## 요약

푸에르토리코는 매 라운드 플레이어들이 역할을 하나씩 선택하고, 선택된 역할의 행동을 모두가 수행하는 고전 경제 전략 게임이다. 농장과 생산 건물을 개척민으로 활성화해 상품을 만들고, 상품을 팔아 돈을 벌거나 배에 실어 승점을 얻으며, 건물을 지어 엔진과 종료 점수를 만든다.

## 기본 정보

- 최소인원: 3명
- 최대인원: 5명
- 추천인원:
  - BGG: 커뮤니티 추천 3-5인, 베스트 4인
  - 보드라이프: 베스트 4인, 추천 3인
- 플레이 시간: 90-150분
- 난이도:
  - BGG: 3.27/5
  - 보드라이프: 3.27
- 메커니즘:
  - BGG: Action Drafting, End Game Bonuses, Follow, Hidden Victory Points, Increase Value of Unchosen Resources, Turn Order: Progressive, Variable Phase Order 등
  - 보드라이프: 단계 순서 변화, 게임 종료 보너스, 비공개 승점, 액션 드래프팅, 차례 순서: 정방향 등

## 룰 핵심

- 정해진 라운드 수는 없고, 종료 조건이 발생할 때까지 진행한다.
- 각 라운드에 플레이어는 총독부터 차례대로 역할 1개를 선택한다.
- 선택된 역할의 행동은 모든 플레이어가 수행할 수 있고, 역할 선택자만 특권을 받는다.
- 선택되지 않은 역할에는 1더블룬이 쌓여 다음 라운드 선택 유인을 만든다.
- 농장/채석장/건물은 개척민이 올라가 있어야 작동한다.
- 상품은 상인으로 팔아 돈을 벌거나 선장으로 선적해 승점을 얻는다.
- 최종 점수는 승점 칩, 건물 승점, 활성화된 대형 건물 보너스로 계산한다.

## 출처별 차이

- 기본판 인원: 2002년 기본판과 보드라이프 `푸에르토리코` 페이지는 3-5인이다. 2011년 10주년판/일부 재판과 `푸에르토리코 1897`은 2-5인 또는 1-5인으로 표시될 수 있으므로 이 문서에서는 기본판과 구분한다.
- 추천인원: BGG는 베스트 4인/추천 3-5인, 보드라이프는 베스트 4인/추천 3인으로 표시한다.
- 난이도: BGG와 보드라이프 모두 3.27로 확인된다.

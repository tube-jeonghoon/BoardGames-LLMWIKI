# 전체 게임 평점 검색 요약

## 요약

현재 위키에 있는 게임 28개의 평점을 BGG 계열 보조 출처와 보드라이프에서 확인해 각 게임 페이지와 룰 페이지의 기본 정보에 `평점` 항목으로 반영했다.

## 확인 방식

- BGG XML API는 인증 요구로 직접 조회하지 못했다.
- BGG 값은 BoardGameMatcher의 `BGG Rating` 공개 표기를 10점 만점으로 환산해 기록했다.
- `윙스팬: 아메리카 확장`은 BoardGameMatcher 페이지를 찾지 못해 BoardGameGeek 검색 스니펫의 평점 표시를 사용했다.
- 보드라이프 값은 각 게임 페이지의 `게임평점` 메타 설명 또는 검색 스니펫을 사용했다.

## 변경한 지식

- 모든 룰 페이지의 기본 정보 표에 `평점` 행을 추가했다.
- 모든 게임 페이지의 기본 정보에 `평점` 항목을 추가했다.
- 하다라의 보드라이프 URL을 현재 게임 페이지인 `game/10861`로 교정했다.

## 모순/주의점

- BGG의 `Average Rating`과 `Geek Rating`은 동일하지 않다. 이번 값은 `BGG Rating`으로 표기했다.
- 보드라이프 평점은 접근 시점에 따라 바뀔 수 있다.
- 확장 게임은 표본 수가 적어 평점 변동성이 더 크다.

# 콩코르디아 룰 검색 요약

## 요약

콩코르디아는 로마 제국을 배경으로 정착민을 이동시키고 도시마다 집을 지어 생산 네트워크를 확장하는 카드 주도 경제 전략 게임이다. 매 턴 손패 카드 1장을 사용해 행동하고, 사용한 카드는 `Tribune`으로 회수한다. 구매한 카드는 행동 선택지를 늘리는 동시에 최종 점수의 신 아이콘 배율이 된다.

## 기본 정보

- 최소인원: 2명
- 최대인원: 5명
- 추천인원:
  - BGG: 커뮤니티 추천 2-5인, 베스트 4인
  - 보드라이프: 베스트 4인, 추천 2인
- 플레이 시간:
  - BGG/보드라이프: 100분
  - PD-Verlag: 90분
  - Rio Grande Games: 45-90분
- 난이도:
  - BGG: 2.99/5
  - 보드라이프: 2.99
- 메커니즘:
  - BGG: Action Retrieval, Advantage Token, Auction: Dutch, Deck, Bag, and Pool Building, End Game Bonuses, Hand Management, Hidden Victory Points, Investment, Movement Points, Open Drafting, Point to Point Movement, Set Collection, Variable Set-up 등
  - 보드라이프: 카드를 플레이해 도시를 확장하고 자원을 관리하는 전략 보드게임

## 룰 핵심

- 턴마다 손패 카드 1장을 내고 그 카드의 행동을 수행한다.
- 사용한 카드는 버림 더미에 쌓이며, `Tribune`으로 회수한다.
- `Architect`로 정착민을 이동시키고 도시마다 집을 지어 생산망을 확장한다.
- `Prefect`와 `Specialist` 카드로 자원을 생산한다.
- `Mercator`로 상품을 사고팔아 돈을 마련한다.
- `Senator`/`Consul`로 새 카드를 구매해 행동과 점수 아이콘을 늘린다.
- 게임 종료는 마지막 카드 구매 또는 15번째 집 건설로 촉발된다.
- 최종 점수는 카드의 신 아이콘별 조건을 모두 계산해 합산한다.

## 출처별 차이

- 추천인원: BGG는 베스트 4인/추천 2-5인, 보드라이프는 베스트 4인/추천 2인으로 표시한다.
- 플레이 시간: BGG/보드라이프는 100분, PD-Verlag는 90분, Rio Grande Games는 45-90분으로 표시한다.
- 연령: BGG/보드라이프는 13세 이상, PD-Verlag 상품 정보는 12세 이상과 13세 이상 표기가 함께 보이고, Rio Grande Games는 14세 이상으로 표시한다.
- 에라타: Rio Grande Games는 초기판 Colonist 카드/룰북 불일치와 일부 Diplomat 카드 비용 오류를 안내한다.

# 버라지 룰 검색 요약

## 핵심 내용

[[버라지]]는 5라운드 동안 기술자를 행동칸에 배치해 댐, 도수관, 발전소를 짓고, 공유 자원인 물의 흐름을 가로채 수력 발전을 하는 고난도 경제/네트워크 게임이다. 건설에 쓴 기술 타일과 기계는 사라지지 않고 건설 휠을 한 바퀴 돌아야 다시 사용할 수 있어, 행동 타이밍과 회수 계획이 핵심이다.

## 새로 추가된 지식

- [[버라지]] 게임 페이지 생성.
- [[버라지 - 룰]] 규칙 페이지 생성.
- 1-4인, 2019년, Simone Luciani/Tommaso Battista 디자인으로 기록.
- 보드라이프 기준 평점 8.2, 난이도 4.12, 언어의존도 `없음`으로 기록.
- BGG 기준 평균 평점 약 8.14, 웨이트 4.11, 플레이 시간 90-180분으로 기록.

## 모순/주의점

- Cranio 영문 제품 페이지는 90분/13세 이상, 이탈리아어 제품 페이지는 120분/14-99세, BGG는 90-180분/14세 이상, 보드라이프는 60-120분/14세 이상으로 표시한다.
- 공식 영문 본판 룰북 PDF는 확인하지 못했다. 공식 이탈리아어 룰북과 한국어/영문 DB 정보를 대조해 작성했다.
- 한국어판 공식 용어, 코리아보드게임즈판 구성, 에라타/정정 스티커 반영 여부는 확인 필요다.
- 레이흐바터르 프로젝트, 5번째 발전회사, 나일강/콜로라도 맵, 레전더리 박스 구성은 별도 페이지로 분리할 만하다.

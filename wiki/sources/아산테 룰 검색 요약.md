# 아산테 룰 검색 요약

## 핵심 내용

[[아산테]]는 2인 전용 카드/경제 게임이다. 각자 가판대에 상품을 모으고, 상품 카드를 매입 또는 판매로 사용해 금을 늘린다. 매 턴 최대 5액션을 카드 뽑기와 카드 사용에 나눠 쓰며, 도구/인물/동물/성지 카드가 거래 효율과 상대 견제를 만든다.

## 새로 추가된 지식

- [[아산테]] 게임 페이지 생성.
- [[아산테 - 룰]] 규칙 페이지 생성.
- 2인 전용, 40분, 2013년, Rüdiger Dorn 디자인, Michael Menzel 아트워크로 기록.
- 보드라이프 기준 언어의존도는 `적당`이며, 카드 텍스트 효과가 많아 한국어판 또는 한글화가 있으면 첫 플레이가 수월하다고 정리.

## 모순/주의점

- 보드라이프는 추천인원에 `추천 1인`도 표시하지만, 규칙상 게임은 2인 전용이다.
- 영어 공식 룰북 원본은 직접 확보하지 못했고, KOSMOS 공식 독일어 룰북 안내와 영어 룰북 전사본을 함께 사용했다.
- 한국어판 실물 룰북과 카드 용어는 확인 필요다.

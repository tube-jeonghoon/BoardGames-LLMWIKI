# 7 원더스 대결 룰 검색 요약

## 요약

7 원더스 대결은 2인 전용 카드 드래프팅/문명 발전 게임이다. 손패를 돌리는 [[7 원더스]]와 달리, 겹쳐 놓인 공개/비공개 카드 구조에서 덮이지 않은 카드 1장을 번갈아 가져간다.

선택한 카드는 건물로 짓거나, 불가사의 건설에 사용하거나, 버리고 코인을 받는다. 군사 말이 상대 수도에 도달하거나 서로 다른 과학 기호 6개를 모으면 즉시 승리하며, 그렇지 않으면 3시대 종료 후 승점으로 승자를 정한다.

## 핵심 확인 사항

- 2인 전용, 30분, 10세 이상.
- BGG/보드라이프 난이도 2.23.
- 카드 사용은 건물 건설, 불가사의 건설, 판매 중 하나.
- 불가사의는 각자 4장씩 가지지만, 게임 전체에서 7개만 건설 가능.
- 자원은 소비되지 않고, 부족한 자원은 은행에서 구매한다.
- 빨간 군사 카드로 충돌 말을 이동하며 수도 도달 시 즉시 승리.
- 서로 다른 과학 기호 6개를 모으면 즉시 과학 승리.

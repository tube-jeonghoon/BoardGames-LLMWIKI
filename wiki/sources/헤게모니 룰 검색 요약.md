# 헤게모니 룰 검색 요약

## 요약

헤게모니는 2-4명이 각기 다른 사회 계층을 맡아 한 국가의 정치와 경제를 시뮬레이션하는 비대칭 카드 주도 전략 게임이다. 게임은 5라운드 동안 진행되며, 각 라운드는 준비, 행동, 생산, 선거, 점수 단계로 나뉜다. 플레이어는 카드 효과 또는 기본 행동으로 회사를 세우고, 노동자를 배치하고, 재화와 서비스를 생산/판매하고, 법안을 발의하며, 정책을 자기 계층에 유리하게 바꾸려 한다.

## 기본 정보

- 최소인원: 2명
- 최대인원: 4명
- 추천인원:
  - BGG: 커뮤니티 추천 2-4인, 베스트 4인
  - 보드라이프: 베스트 4인, 추천 3인
- 플레이 시간: 90-180분
- 난이도:
  - BGG: 4.25/5
  - 보드라이프: 4.23
- 메커니즘:
  - BGG: Action / Event, Hand Management, Simulation, Variable Player Powers, Voting, Worker Placement, Different Worker Types 등
  - 보드엠/공식 소개: 카드 주도, 비대칭, 정치-경제 전략

## 룰 핵심

- 2인: 노동자 계층과 자본가 계층.
- 3인: 노동자, 중산층, 자본가.
- 4인: 노동자, 중산층, 자본가, 정부.
- 각 플레이어는 행동 단계에서 카드 5장을 사용하고 2장을 다음 라운드로 남긴다.
- 카드는 효과로 쓰거나 기본 행동으로 버릴 수 있다.
- 생산 단계에서 회사는 임금을 지급하고 상품/서비스를 생산한다.
- 노동자와 중산층은 식량 등 기본 필요를 충족해야 한다.
- 정책은 임금, 세금, 복지, 무역, 이민 등 여러 규칙에 직접 영향을 준다.
- 발의된 법안은 선거 단계에서 투표로 통과 여부가 결정된다.
- 각 계층은 서로 다른 방식으로 VP를 얻는다.

## 출처별 차이

- 추천인원: BGG는 베스트 4인과 추천 2-4인으로 해석 가능하지만, 보드라이프는 베스트 4인/추천 3인으로 표시한다.
- 난이도: BGG 4.25, 보드라이프 4.23으로 거의 같지만 수치는 별도 기록한다.
- 국내 자료: BoardM Factory는 한국어 규칙서 v1.4, 참조표 v1.2, FAQ/정오표, 카드 에라타가 있음을 안내한다. 실제 세부 용어는 한국어 PDF 원문 확인 시 추가 대조 필요.

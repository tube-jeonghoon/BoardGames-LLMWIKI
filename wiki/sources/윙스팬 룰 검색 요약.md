# 윙스팬 룰 검색 요약

## 요약

윙스팬은 조류 보호구역에 새를 배치하고 먹이, 알, 카드 생산 엔진을 키우는 카드 중심 엔진 빌딩 게임이다. 숲/초원/습지 3개 서식지에 새를 놓고, 각 서식지 행동을 할 때 해당 줄의 새 능력을 오른쪽에서 왼쪽으로 활성화한다.

게임은 4라운드로 진행되며, 라운드 목표 점수 표시 때문에 사용할 수 있는 행동 큐브가 매 라운드 줄어든다. 4라운드 종료 후 새 카드 점수, 보너스 카드, 라운드 목표, 알, 저장 먹이, 끼워 넣은 카드를 합산한다.

## 핵심 확인 사항

- 기본판 기준 1-5명.
- BGG: 추천 1-4인, 베스트 3인 / 보드라이프: 베스트 3인, 추천 2인.
- 4라운드 진행, 행동 수는 8/7/6/5.
- 주요 행동은 새 놓기, 먹이 얻기, 알 낳기, 새 카드 뽑기.
- 새 놓기 비용은 먹이 비용과 열에 따른 알 비용.
- 갈색 능력은 서식지 행동 때 오른쪽에서 왼쪽으로 활성화한다.
- 최종 점수는 새, 보너스, 라운드 목표, 알, 저장 먹이, 끼워 넣은 카드.

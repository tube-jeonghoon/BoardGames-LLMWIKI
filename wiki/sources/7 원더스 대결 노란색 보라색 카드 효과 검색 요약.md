# 7 원더스 대결 노란색 보라색 카드 효과 검색 요약

## 핵심 내용

[[7 원더스 대결]] 기본판의 노란색 상업 건물과 보라색 길드 카드 효과를 정리했다. 노란색 카드는 코인, 거래 할인, 선택형 자원, 조건부 점수로 경제 기반을 만들고, 보라색 길드는 시대 III에 일부만 등장해 양쪽 도시의 카드 색상/불가사의/코인 상태를 비교해 점수를 준다.

## 새로 추가된 지식

- [[7 원더스 대결 - 노란색과 보라색 카드]] 페이지 생성.
- 노란색 카드 효과를 시대별로 정리.
- 보라색 길드 7장의 효과와 계산 주의점을 정리.
- `건설 시 코인은 한 번만 받음`, `길드 점수 기준 도시는 코인 획득 때와 달라도 됨`, `노란 카드 생산 자원은 상대 거래 비용 증가에 포함하지 않음`을 강조.
